# 注释

### 变量

- {"data":3.0,"secondvalue":0,"value":0,"varname":""}

- 变量/绑定变量

- "secondvalue":0

- 表示是否为绑定变量；int类型；[0,1]
- 1:绑定变量；格式：GD[GI0002]

- "value":0

- 表示数据是什么类型；int类型
- 0：表示数值型，此时数值填"data"节点中；7：此时是字符串类型，字符串填"varname"节点中；8：此时使用变量或绑定变量，变量填"varname"节点中

- "data":3.0

- 数值型在此显示

- "varname":""

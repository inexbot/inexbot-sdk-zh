# 指令-网络通讯类

### 发送数据

- 说明：
- 向个网络设备发送字符串信息或者变量值
- 发送数据
- 指令名：SENDMSG
- "ID":7
- 工艺号；int类型；取值范围[1-9]
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "sendChar":"UUUUUU"
- 发送的字符消息；string类型
- "type":94
- 发送数据在枚举数列type中为94；int类型
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
  "ID":7,
  "logout":false,
  "sendChar":"UUUUUU",
  "type":94,
  "userParamInt":0,
  "userParamString":""
}
```
### 解析数据

- 二次开发中客户自定义，目前无意义
- 说明：
- 该指令用来解析外部设备传来的一组数据，并将数据存入多个变量
- 解析数据
- 指令名：PARSEMSG
- "ID":3
- 工艺号；int类型；取值范围[1-9]
- "clearCache":1
- 解析后是否清楚缓存区；int类型；取值范围[0，1]
- 0：否，1：是
- "dataStorage":{"data":0.0,"secondvalue":0,"value":4,"varname":"GI009"}
- 记录提取数据的数量；手填值int类型0或int类型变量，全局或局部
- 手填值0表示不记录提取数据的数量
- "dataVariable":{"data":0.0,"secondvalue":1,"value":5,"varname":"GD[GI001]"}
- 查询的数据依次存放的首位置；int类型或double类型或string类型变量，全局或局部
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":96
- 解析数据在枚举数列type中为96；int类型
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
"ID":3,
"clearCache":1,
"dataStorage":{"data":0.0,"secondvalue":0,"value":4,"varname":"GI009"},
"dataVariable":{"data":0.0,"secondvalue":1,"value":5,"varname":"GD[GI001]"},
"logout":false,
"type":96,
"userParamInt":0,
"userParamString":""
}
```
### 读取

- 二次开发中客户自定义，目前无意义
- 说明：
- 读取通过以太网或Modbus通讯发送点位，并将点位存入到变量
- 读取
- 指令名：READCOMM
- "ID":6
- 工艺号；int类型；取值范围[1-9]
- "commType":1
- 通讯方式；int类型；取值范围[0,1]
- 0：ETHERENT通讯，1：MODBUS通讯
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "m_varname":"I001"
- 点位存放个数；int类型变量，全局或局部
- "positionId":"GP0001"
- 点位存放首个变量；点位信息类型P或GP
- P：点位局部变量，GP：点位全局变量
- "type":110
- 读取在枚举数列type中为110；int类型
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
  "ID":6,
  "commType":1,
  "logout":false,
  "m_varname":"I001",
  "positionId":"GP0001",
  "type":110,
  "userParamInt":0,
  "userParamString":""
}
```

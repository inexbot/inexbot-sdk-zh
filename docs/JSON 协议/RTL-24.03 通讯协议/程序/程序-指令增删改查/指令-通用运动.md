# 指令-通用运动

### 点到点运动指令

- 说明：
- 点到点运动为关节运动，单位为%
- 点到点运动的指令
- 指令名：MOVJ
- "ParaACC":{"data":3.0,"secondvalue":0,"value":0,"varname":""},
- 加速度变量;doub类型；取值范围[1,100]；单位%
- "ParaDEC":{"data":4.0,"secondvalue":0,"value":0,"varname":""},
- 减速度变量；double类型；取值范围[1,100]；单位%
- "ParaV":{"data":1.0,"secondvalue":0,"value":0,"varname":"","m_vUnit":2}
- 速度变量；double类型；取值范围[1,100]；单位%
- 除m_vUnit外，与普通变量一致
- "m_vUnit":2,
- 速度单位：0=cm/s 1=mm/s 2=百分比；
- 此时只能使用2，关节运动使用百分比
- "ParaPL":{"data":2.0,"secondvalue":0,"value":0,"varname":""},
- 平滑变量；double类型；取值范围[0,5]
- "ParaTIME":{"data":5.0,"secondvalue":0,"value":0,"varname":""},
- 提前执行时间变量；double类型；取值范围自然数
- "RobotPos":{}
- 点位信息
- 当使用绑定变量点位时，不使用该节点
- "type":1
- 1：表示该指令为MOVJ指令；int类型
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "posidname":"GP$GI001"
- 绑定变量时的点位名；string类型
- "positionId":"P0001"
- 非绑定变量时的点位名；string类型
- "posidtype":1
- 点位是否为绑定变量；0或1
- 1：绑定变量，0：非绑定变量
- 备注：

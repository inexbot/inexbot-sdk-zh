# 指令-打磨工艺类

### 棱边打磨

- 说明：
- 棱边打磨
- 指令名：POLISH_EDGE
- "ParaACC":{"data":0.0,"secondvalue":0,"value":1,"varname":"I002"}
- 加速度；double类型手填值或double类型或int类型全局或局部变量或绑定变量；取值范围[1,100]%
- "ParaAHEAD":{"data":0.0,"secondvalue":1,"value":1,"varname":"GD[I001]"}
- 含义与"aheadtype"有关；double类型手填值或double类型或int类型全局或局部变量或绑定变量；取值范围[1,100]%
- "ParaDEC":{"data":0.0,"secondvalue":0,"value":1,"varname":"GI008"}
- 减速度；double类型手填值或double类型或int类型全局或局部变量或绑定变量；取值范围[1,100]%
- "ParaPL":{"data":0.0,"secondvalue":1,"value":1,"varname":"I[GI001]"}
- 平滑；int类型手填值或int类型全局或局部变量或绑定变量；取值范围[0,5]
- "ParaTIME":{"data":0.0,"secondvalue":1,"value":1,"varname":"GI[I080]"}
- 提前执行；double类型手填值或double类型或int类型全局或局部变量或绑定变量
- "ParaV":{"data":0.0,"m_vUnit":1,"secondvalue":0,"value":1,"varname":"I001"}
- 速度；double类型手填值或double类型或int类型全局或局部变量或绑定变量；取值范围[1,1000]mm/s
- "aheadtype":2
- 0:提前时间；1:提前距离；2:提前进度
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "polish":89
- 打磨次数；int类型；取值范围[1,99]
- "polishAngle":70.0
- 角度；double类型；取值范围[-180,180]度
- "polishID":30
- 打磨工艺号；int类型；取值范围[1,99]
- "posidname":"GP$GI001"
- 点位；P或GP绑定变量
- "posidtype":1
- 点位类型表示参数；int类型；取值范围[0,1]
- 0:点位变量；1:点位绑定变量
- "positionId":""
- 点位；P或GP变量
- "posidtype"节点为0时使用
- "type":109
- POLISH_EDGE在枚举数列type中为109
- "userParamInt":0

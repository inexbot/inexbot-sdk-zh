# 指令-运动控制类

### 外部轴运动

- 说明：
- 与外部轴有关的运动指令
- 必须是在存在外部轴的前提下才可以使用
- 外部轴点到点运动的指令
- 指令名：MOVJEXT
- "ParaACC":{"data":3.0,"secondvalue":0,"value":0,"varname":""}
- 加速度;doub类型；取值范围[1,100]；单位%
- "ParaDEC":{"data":4.0,"secondvalue":0,"value":0,"varname":""}
- 减速度；double类型；取值范围[1,100]；单位%
- "ParaEVJ":{"data":0.0,"secondvalue":1,"value":1,"varname":"GI[I001]"}
- 外部轴速度调整比率；double类型；取值范围[1~100]
- 值为字符串“0”时表示不使用；即"value":1，"varname":"0"
- "ParaV":{"data":1.0,"m_vUnit":2,"secondvalue":0,"value":0,"varname":""}
- 速度；double类型；取值范围[1，100]；单位%
- 除m_vUnit外同上
- "m_vUnit":2
- 速度单位：0=cm/s 1=mm/s 2=百分比；此时只能使用2
- "ParaPL":{"data":2.0,"secondvalue":0,"value":0,"varname":""}
- 平滑；double类型；取值范围[0,5]
- "ParaTIME":{"data":5.0,"secondvalue":0,"value":0,"varname":""}
- 提前执行时间；double类型；取值范围自然数
- "RobotPos":{}
- 点位信息
- 当使用绑定变量点位时，不使用该节点
- "type":33
- 33：表示该指令为MOVJEXT指令；int类型
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "posidname":"GP$GI001"
- 绑定变量时的点位名
- "positionId":"P0001"
- 非绑定变量时的点位名
- "posidtype":1

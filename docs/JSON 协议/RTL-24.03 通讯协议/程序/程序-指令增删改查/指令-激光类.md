# 指令-激光类

### 激光开始

- 说明：
- 激光开始
- 指令名：LASER_ON
- "ID":5
- 工艺号；int类型；取值范围[1,9]
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":60
- VISION_RUN在枚举数列type中为60
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
  "ID":5,
  "logout":false,
  "type":60,
  "userParamInt":0,
  "userParamString":""
}
```

### 切割圆

- 二次开发中客户自定义，目前无意义
- 说明：
- 切割圆
- 指令名：LASER_CIRCLE
- "ParaACC":{"data":0.0,"secondvalue":1,"value":1,"varname":"GI[GI001]"}
- 加速度；double手填值或int类型或double类型全局或局部变量或绑定变量；取值范围[1,100]%
- "ParaDEC":{"data":0.0,"secondvalue":1,"value":1,"varname":"GD[I001]"}
- 减速度；double手填值或int类型或double类型全局或局部变量或绑定变量；取值范围[1,100]%
- "ParaTIME":{"data":0.0,"secondvalue":1,"value":1,"varname":"D[GI001]"}
- 提前运行时间；加速度；double手填值或int类型或double类型全局或局部变量或绑定变量；ms
- "ParaV":{"data":0.0,"m_vUnit":1,"secondvalue":0,"value":1,"varname":"D001"},
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "margin":360.0
- 补差；double类型；取值范围[0,500]mm
- "posidname":"GP$I001"
- 中心点；P或GP变量或绑定变量
- "posidtype":1
- 点位是否为绑定变量；0或1
- 1：绑定变量，0：非绑定变量
- "radius":28.0
- 半径；double类型；取值范围[1,3000]mm
- "type":62
- LASER_CIRCLE在枚举数列type中为62
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

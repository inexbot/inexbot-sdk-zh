# 指令-定时器类

### 延时

- 说明：
- 延时设置的值，到达设置的时间后程序继续运行
- 延时
- 命令字：TIMER
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "paraTime":{"data":500.0,"secondvalue":0,"value":0,"varname":""}
- 延时持续时间；手填double类型或int类型或double类型变量；取值范围（0,9999]
- 单位s
- "type":7
- TIMER在枚举数列type中为7；int类型
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
  "logout":false,
  "paraTime":{"data":500.0,"secondvalue":0,"value":0,"varname":""},
  "type":7,
  "userParamInt":0,
  "userParamString":""
}
```
- 二次开发中客户自定义，目前无意义

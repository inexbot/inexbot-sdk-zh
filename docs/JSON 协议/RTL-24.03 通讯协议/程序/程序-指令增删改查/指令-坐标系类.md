# 指令-坐标系类

### 切换工具手

- 说明：
- 切换工具手，0为无工具手
- 切换工具手
- 指令名：SWITCHTOOL
- "_variable":{"data":0.0,"secondvalue":0,"value":4,"varname":"GI009"}
- 变量类型；手填int类型或int类型，全局或局部；取值范围[0,999]
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":48
- 切换工具手在枚举数列type中为48
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
  "_variable":{"data":0.0,"secondvalue":0,"value":4,"varname":"GI009"}
  "logout":false,
  "type":48,
  "userParamInt":0,
  "userParamString":""
}
```

### 用户坐标转换

- 二次开发中客户自定义，目前无意义
- 说明：
- 用户坐标转换
- 将B、C用户坐标系叠加（×），结果置入A用户坐标系
- 切换工具手
- 指令名：USERCOORD_TRANS
- "CoordA":8
- 用户坐标A；int类型；取值范围[1-999]整数
- 结果存入该用户坐标系，这里是用户坐标系序号
- "CoordB":18
- 用户坐标B；int类型；取值范围[1-999]整数
- "CoordC":18
- 用户坐标C；int类型；取值范围[1-999]整数
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":49
- 切换用户坐标在枚举数列type中为49
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
"CoordA":8,
"CoordB":18,
"CoordC":28,
"logout":false,
"type":117,
"userParamInt":0,
"userParamString":""
}
```

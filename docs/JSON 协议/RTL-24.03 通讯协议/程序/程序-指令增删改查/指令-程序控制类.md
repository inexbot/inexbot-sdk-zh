# 指令-程序控制类

### 开启线程

- 说明：
- 开启全局后台或者局部后台任务
- 开启线程的指令
- 指令名：PTHREAD_START
- "jobName":"SFDFF"
- 后台任务程序文件名；string类型
- "localOrGlobal":0
- 后台类型；int类型
- 0：局部后台；1：全局后台
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":90
- PTHREAD_START 在枚举数列type中为90
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
    "jobName":"SFDFF",
    "localOrGlobal":0,
    "logout": false,
    "type": 90,
    "userParamInt": 0,
    "userParamString": ""
}
```

### 弹窗指令

- 二次开发中客户自定义，目前无意义
- 说明：
- 运行指令时弹出所填写的提示内容窗口，按钮数量为选项数量，点击按钮会将变量值存入绑定的变量
- 停止运行的指令
- 指令名：WINDOW
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":208
- WINDOW在枚举数列type中为208
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "count":3
- 选项数量；int类型；[0,3]
- "text1":"AAAAA"
- 提示内容;string类型
- "text2":"1"
- 选项1内容；string类型
- "text3":"2"
- 选项2内容；string类型
- "text4":"3"
- 选项3内容；string类型
- "value1":1111
- 选项1对应变量值；int类型
- "value2":2222
- 选项2对应变量值；int类型
- "value3":3333
- 选项3对应变量值；int类型
- "variable":{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}

```json
{
    "logout": false,
    "type": 208,
    "userParamInt": 0,
    "userParamString": "",
    "count": 3,
    "text1": "AAAAA",
    "text2": "1",
    "text3": "2",
    "text4": "3",
    "value1": 1111,
    "value2": 2222,
    "value3": 3333,
    "variable": {
        "data": 0.0,
        "secondvalue": 0,
        "value": 1,
        "varname": "I001"
    }
}
```
### 线程状态

- 绑定变量
- 说明：
- 查看当前所执行的线程程序的状态，停止等于1，暂停等于2，运行等于3
- 线程状态的指令
- 指令名：PTHREAD_STATE
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":209
- PTHREAD_START 在枚举数列type中为209
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "jobName":"SFDFF"
- 当选取后台类型时 选择的程序名；string类型
- "localOrGlobal":0
- 程序类型；int类型；取值范围[0,2]
- 0：局部后台；1：全局后台；2：主程序
- "m_VariableValue":{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}

```json
{
    "logout": false,
    "type": 209,
    "userParamInt": 0,
    "userParamString": ""
    "jobName": "SFDFF",
    "localOrGlobal": 0,
    "m_VariableValue": {
        "data": 0.0,
        "secondvalue": 0,
        "value": 1,
        "varname": "I001"
    },
}
```

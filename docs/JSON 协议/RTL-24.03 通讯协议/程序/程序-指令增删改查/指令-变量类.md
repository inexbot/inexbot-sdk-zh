# 指令-变量类

### 赋值

- 说明：

- 给定义的整型、浮点型、布尔型和字符串变量赋值

- 字符串追加指令

- 指令名：SET

- "logout":false

- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行

- "type":251

- 退出 在枚举数列type中为251

- "userParamInt":0

- 二次开发中客户自定义，目前无意义

- "userParamString":""

- 二次开发中客户自定义，目前无意义

- "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"}

- 变量；int类型或bool类型或double类型或string类型变量，全局或局部

- "variableValue":{"data":0.0,"secondvalue":0,"value":8,"varname":"GD009"}

```
{
  "logout":false,
  "type":41,
  "userParamInt":0,
  "userParamString":"",
  "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"},
  "variableValue":{"data":0.0,"secondvalue":0,"value":8,"varname":"GD009"}
}
```
### 写入文件

- 变量值；手填值或bool类型或int类型或double类型变量

- 说明：

- 将缓存的数据存入硬盘。
- 在程序运行过程中，所有的计算、赋值操作均是对缓存中的数值进行更改的，并不会存入系统文件中。若要强制将内存中的全局数值变量写入文件中，则可以使用FORCESET指令

- 字符串追加指令

- 指令名：FORCESET

- "logout":false

- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行

- "type":251

- 退出 在枚举数列type中为251

- "userParamInt":0

- 二次开发中客户自定义，目前无意义

- "userParamString":""

- 二次开发中客户自定义，目前无意义

- "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"GI001"}

```
{
  "logout":false,
  "type":42,
  "userParamInt":0,
  "userParamString":"",
  "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"GI001"}
}
- 变量名；int类型或bool类型或double类型，全局
```

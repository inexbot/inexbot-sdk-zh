# 指令-视觉命令类

### 视觉开始

- 说明：
- 执行视觉开始指令后控制器与相机连接
- 视觉开始
- 指令名：VISION_RUN
- "ID":14
- 工艺号；int类型；取值范围[1,99]
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":79
- VISION_RUN在枚举数列type中为79
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""

```json
{
  "ID":14,
  "logout":false,
  "type":79,
  "userParamInt":0,
  "userParamString":""
}
```

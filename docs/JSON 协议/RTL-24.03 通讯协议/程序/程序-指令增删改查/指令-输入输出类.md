# 指令-输入输出类

### IO输入

- 说明：
- 输入是相对于控制器而言，由外部到控制器
- IO输入的指令
- 指令名：DIN
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":17
- DIN 在枚举数列type中为17
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "ParaGroupNum":{"data":2.0,"secondvalue":0,"value":0,"varname":""}
- 输入组号；int类型
- 取值范围和一共多少组有关，IO板值*16/几路输入
- "ParaGroupValue":{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}
- 端口值存入；变量类型，全局或局部的bool或int型变量
- "groupType":"IGH#"

```json
{
    "logout": false,
    "type": 17,
    "userParamInt": 0,
    "userParamString": "",
    "ParaGroupNum":{"data":2.0,"secondvalue":0,"value":0,"varname":""},
    "ParaGroupValue":{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"},
    "groupType":"IGH#"
}
```
### IO输出

- din类型；string类型
- 1路输入：IN# ，4路输入：IGH# ，8路输入： IG#
- 说明：
- 将IO板上对应的IO端口置高或置低
- IO输出的指令
- 指令名：DOUT
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":16
- DOUT 在枚举数列type中为16
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "ParaGroupNum":{"data":2.0,"secondvalue":0,"value":0,"varname":""}
- 输入组号；int类型
- 取值范围和一共多少组有关，IO板值*16/几路输出
- "ParaGroupValue":{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}
- 端口值输出；变量类型，全局或局部的bool或int型变量
- "groupType":"OT#"
- din类型；string类型
- 1路输入：OT#，4路输入： OGH# ，8路输入： OG#
- "errorHanding":1
- 1:计时结束停止，0：输出值保持
- "ParaGroupTime":{"data":3.0,"secondvalue":0,"value":0,"varname":""}

```json
{
    "logout": false,
    "type": 16,
    "userParamInt": 0,
    "userParamString": "",
    "ParaGroupNum":{"data":2.0,"secondvalue":0,"value":0,"varname":""},
    "ParaGroupValue":{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"},
    "errorHanding":1,
    "groupType":"IGH#",
    "ParaGroupTime":{"data":3.0,"secondvalue":0,"value":0,"varname":""}
}
```

### 脉冲输出

- 模拟输入口；int类型；取值范围1-IO板数*2
- 说明：
- 按照设定的脉冲频率与个数，在R1 PWMIO板上的DB9端子的引脚4（PWM+）上进行输出
- 脉冲输出的指令
- 指令名：PULSEOUT
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":22
- PULSEOUT在枚举数列type中为22
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "rate":10
- 频率;int类型；[1,1000000]
- "sum":100

```json
{
    "rate":10,
    "logout": false,
    "type": 22,
    "sum":100,
    "userParamInt": 0,
    "userParamString": ""
}
```

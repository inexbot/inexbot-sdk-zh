# 指令-字符串类

### 字符串追加

- 说明：
- 在原有字符串变量或者新的字符变量中追加需要的字符，就构成了一个新的字符串变量
- 字符串追加指令
- 指令名：STRING_SPELL
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":251
- 退出 在枚举数列type中为251
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}
- 变量类型及名称；string类型变量，全局或局部变量
- "variableValue":{"data":0.0,"secondvalue":0,"value":7,"varname":"11111"}

```json
{
  "logout":false,
  "type":223,
  "userParamInt":0,
  "userParamString":"",
  "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"},
  "variableValue":{"data":0.0,"secondvalue":0,"value":8,"varname":"GS001"}
}
```
### 索引截取

- 常数或绑定其他变量；string类型手填值或变量，全局或局部
- 说明：
- 截取一个字符串变量中的其中一部分字符串，并把截取到部分字符串存到指定的变量
- 索引截取的指令
- 指令名：STRING_SLICE
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":224
- STRING_SLICE 在枚举数列type中为224
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "startIndex":{"data":2.0,"secondvalue":0,"value":0,"varname":""}
- 起始索引位置；int类型手填值或者变量，全局或局部
- 1：开始的位置
- "endIndex":{"data":8.0,"secondvalue":0,"value":0,"varname":""}
- 结束索引位置；int类型手填值或者变量，全局或局部
- "dataVariable":{"data":0.0,"secondvalue":1,"value":8,"varname":"GS[I001]"}
- 查询的数据依次存放的位置；string类型变量，全局或局部
- "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}

```json
{
  "logout":false,
  "type":224,
  "userParamInt":0,
  "userParamString":"",
  "startIndex":{"data":2.0,"secondvalue":0,"value":0,"varname":""},
  "endIndex":{"data":8.0,"secondvalue":0,"value":0,"varname":""},
  "dataVariable":{"data":0.0,"secondvalue":1,"value":8,"varname":"GS[I001]"},
  "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}
}
```

### 字符串定位

- 被提取的字符串变量名；string类型变量，全局或局部
- 说明：
- 查询字符串变量里面的一种字符所在的位置，并把位置和数量存到指定变量
- 如果待索引变量在一个字符串里面是不连续的字符，在执行定位查询指令时，字符的位置和个数是无法读取出来的
- 如果待索引变量在一个字符串里面是连续的字符，在执行定位查询指令时，读取出来的字符位置是第一个索引变量的位置,数据存放个数为1
- 字符串定位的指令
- 指令名：STRING_LOCATE
- "dataStorage":{"data":0.0,"secondvalue":1,"value":8,"varname":"GI[GI001]"}
- 记录提取数据的数量；string类型的手填“0”或者int类型变量，全局或局部
- string类型的手填“0”表示不使用
- "dataVariable":{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"},
- 查询的数据依次存放的首位置；int类型变量，全局或局部
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "startIndex":{"data":0.0,"secondvalue":0,"value":7,"varname":"RRRR"}
- 需要搜索的字符；string类型手填值或变量，全局或局部
- "type":226
- 字符串定位在枚举数列type中为226
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}

```json
{
  "dataStorage":{"data":0.0,"secondvalue":1,"value":8,"varname":"GI[GI001]"},
  "dataVariable":{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"},
  "logout":false,
  "startIndex":{"data":0.0,"secondvalue":0,"value":7,"varname":"RRRR"},
```

### 字符串转非字符串

- 待计算长度的变量；string类型变量，全局或局部
- 说明：
- 字符串变量转换成非字符串变量
- 字符串变量转换成非字符串变量时只会将字符串变量的数值部分转译到非字符串变量，在转译过程中会从字符串变量的第一个字符开始转译，直到检测到的字符是非数值的字符时转译结束，将检测到的数值部分的字符转译到非字符变量，如果检测到的第一个字符就是非数值的字符，那字符串无法转译成功
- 字符串转非字符串的指令
- 指令名：STRING_TO
- "nonVariable":{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"}
- 转译的目标变量；int类型或double类型或bool类型变量，局部或全局
- "logout":false
- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行
- "type":228
- 字符串转非字符串在枚举数列type中为228
- "userParamInt":0
- 二次开发中客户自定义，目前无意义
- "userParamString":""
- 二次开发中客户自定义，目前无意义
- "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}

```json
{
  "nonVariable":{"data":0.0,"secondvalue":0,"value":8,"varname":"I001"},
  "logout":false,
  "type":228,
  "userParamInt":0,
  "userParamString":"",
  "variable":{"data":0.0,"secondvalue":0,"value":8,"varname":"S001"}
}
```

# 作业文件传输

#### 示教器发送作业文件指令的时候，发送下面命令：

(用于无作作业文件指令运行，向作业文件里插入指令，请使用0x3121)

*0x3001 FILE_INSTRUCTION

{

"robot":1,		//机器人号

cmd->toJson()		//指令json列表

}

### 作业文件收发过程

#### 一：控制器中的作业文件少于示教器，则需要示教器向控制器上传作业文件

##### 1.示教器向控制器发送获取控制器ID指令0x5052

*0x5052  IDENTIFY_NUMBER_INQUIRE

控制器向示教器发送控制器ID指令0x5053

*0x5053IDENTIFY_NUMBER_RESPOND

{

"controllerID":"4DC9F0249098C82E"//控制器ID

}

获取到连接的控制器ID之后，在同步作业文件时需判断此ID是否已知。已知则判断路

径”dir/controllerID”是否存在，若存在则不做操作；若不存在则创建工作文件路径，格

式：”dir/controllerID/robotJob/R1”。多个机器人则表示R1-4。

##### 2.示教器向控制器发送指令0x5532

示教器请求获取作业文件列表

*0x5532JOBFILE_LIST_INQUIRE

控制器回复作业文件列表指令0x5533

*0x5533JOBFILE_SUM_RESPOND

{

"absolutepath":{"/job/R1/","/job/R2/","/job/R3/","/job/R4/"},//作业文件的文件夹列表

"jobfilenum":{2,33,233,666}//作业文件夹中包含的文件个数

}

此时开启停止下载计时器，设定5秒，作用为判断下载文件过程中是否等待超时

##### 3.控制器主动向示教器发送指令0x5534

*0x5534JOBFILE_LIST_RESPOND

{

"robot":1,

"listnum":2,//最多10个

"jobfilelist":

[

{

"name":"xxx.JBR",//作业文件名字

"MD5":"123"//MD5值作用是判断文件是否相同

},

{

"name":"yyy.JBR",

"MD5":"132"

},

...

]

}

控制器回复的参数作用是于本地的作业文件作比较。若文件相同，判断MD5值是否相同，

若MD5值相同则不做处理，不同则需要删除本地文件并重新下载。记录需要下载的文件：

文件名和路径存储在下载作业文件列表中(加绝对路径)，若文件不同，也需记录需要被下载

的文件到列表中。

##### 4.关于文件下载与上传，需要示教器连接5000端口进行文件传输(指令等命令传输走6000

端口)

##### 5.示教器与控制器文件传输网络已连接，开始进行文件传输

##### 6.示教器发送需要上传的作业文件的名字与大小0x5501

*0x5501REQUEST_UPLOAD_FILE

{

"name":"vxWorks",//示教器发送文件的文件名

"size":4096//示教器发送文件的大小

}

示教器开始请求上传文件，等待控制器回复

##### 7.控制器回复是否同意上传0x5502

*0x5502REQUEST_UPLOAD_RESPOND

{

"answer":"yes"//"yes"表示同意上传文件请求，"busy"表示控制器忙碌

}

##### 8.示教器接受“yes”开始发送文件指令0x5512

*0x5512

文件写入二进制流，采取切片循环发送，若剩余文件大小超过规定数值（2*1024），则每次

发送规定数据大小，直到剩余文件大小小于规定的每次发送数据大小，将剩余文件数据发送

完毕。

##### 9.控制器接收文件完毕，发送指令0x5513

*0x5513RECEIVE_UPLOAD_FINISH

{

“finish”:true//等待下载计时器停止，导入新上传文件

}

##### 10.若有多个作业文件需要同步到控制器中则需要重复6~9步

#### 二：控制器中的作业文件多于示教器，则需要示教器向控制器下载作业文件

1~5上传下载一样通信

##### 6.示教器开始发送列表中的作业文件名称0x5504

*0x5504

{

"name":"job/R1/WWWWW.JBR"//需要下载的作业文件名称

}

参数name是作业文件在控制器中的作业文件路径加上名字。在0x5533和ox5534中

控制器有向示教器发送路径和名字。示教器提示开始同步作业文件。

##### 7.控制器回复0x5505

*0x5505

{

"answer":"yes"//控制器是否同意下载

"name":"job/R1/WWWWW.JBR"//需要被下载的作业文件名称

"size":217//此作业文件的大小

}

##### 8.控制器主动发送0x5522

*0x5522

##### 9.查看示教器是否接收参数成功，示教器发送0x5523

*0x5523

{

"finish":true//示教器接收成功

}

以上过程代表一个作业文件下载完成。若有多个作业文件需要下载，则在0x5534中存

入的列表减一，继续重复以上过程。

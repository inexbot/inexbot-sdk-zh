# 自启动程序运行

#### 示教器设置：

*0x5081   SELF_START_RUN_SET

```json
{
"robot":1,       //机器人号
"enable":false,   //是否使能
"count":1,       //运行次数
"speed":15,      //运行速度
“jobname":"Q1"  //作业文件名字
}
```

#### 查询：

*0x5082    SELF_START_RUN_INQUIRE

```json
{
"robot":1       //机器人号
}
```

#### 控制器返回：

*0x5083    SELF_START_RUN_RESPOND

```json
{
"robot":1,       //机器人号
"enable":false,   //是否使能
"count":1,       //运行次数
"speed":15,      //运行速度
“jobname":"Q1"  //作业文件名字
}
```

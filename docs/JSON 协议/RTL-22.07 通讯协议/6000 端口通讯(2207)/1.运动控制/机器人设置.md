# 机器人设置

### 点动速度设置

#### 设置关节轴点动速度时，发送：

*0x2604 JOG_JOINTPARAMETER_SET

```json
{
"AxisNum":1, //关节轴编号
"minTrajectTime":
{
"minAccTime":0.10,   //最小加速度时间
"minDecTime":0.10， //最小减速度时间
},
"MaxSpeed":10, //关节轴最大点动速度 单位：度°/s
"MaxAcc":10, //关节轴点动加速度 单位：度°/s
}
```

#### 查询关节轴点动速度时，发送：

*0x2605 JOG_JOINTPARAMETER_INQUIRE

```json
{
"AxisNum":1, //关节轴编号
}
```

#### 收到查询关节点动速度时，返回：

*0x2606 JOG_JOINTPARAMETER_RESPOND

```json
{
"AxisNum":1, //关节轴编号
"MaxSpeed":10, //关节轴最大点动速度 单位：度°/s
"MaxAcc":10, //关节轴点动加速度 单位：度°/s
}
```

#### 设置直角坐标点动速度时，发送：

*0x2607 JOG_RECTPARAMETER_SET

```json
{
"MaxSpeed":10,  //关节轴最大点动速度 单位：mm/s
"MaxAcc":10,    //关节轴点动加速度 单位：mm/s2
}
```

#### 查询直角坐标节轴点动速度时，发送：

*0x2608 JOG_RECTPARAMETER_INQUIRE

data:空

#### 收到查询直角坐标点动速度时，返回：

*0x2609 JOG_RECTPARAMETER_RESPOND

```json
{
"MaxSpeed":10, //关节轴点动加速度 单位：mm/s
"MaxAcc":10, //关节轴最大点动速度 单位：mm/s2
}
```

### 当前位置获取

当前位置模式有：

—控制器当前坐标：-1

—关节坐标(Joint)：0

—直角坐标(Cart)：1

—工具坐标(Tool)：2

—用户坐标(User)：3

#### 示教器需要获取当前位置的时候，发送下面命令：

*0x2A02 CURRENTPOS_INQUIRE

```json
{
"robot":1
"coord":2
}
```json
"robot":	表示机器人1，2，3，4

"coord":   坐标模式 ： 1-控制器当前坐标 0-关节坐标 1-直角坐标 2-工具坐标  3-用户坐标 4-电机位置

#### 控制器收到查询命令时，发送下面命令：

*0x2A03 CURRENTPOS_RESPOND

- 命令字：0x2304 DEADMAN_MODE_SET

```json
{
    "robot": 1,
    "deg": 1,
    "pos": [0, 0.1, 2, 3.3, 44, 555.55, 0.0],
    "posDeg": [0.0, 0.0, 1, 2, 3.3, 44, 5.55],
    "coord": -1,
    "configuration": 1
}
```json
"robot":	表示机器人1，2，3，4

"deg":      是否为ACS坐标系；0：是；1：不是

"pos":[0, 0.1, 2, 3.3, 44, 555.55,0.0], //弧度点位值 坐标值用7位存储关节坐标下

前六位为J1~J6轴角度值，单位：度 ，直角、工具、用户坐标前三位为XYZ坐标值，单位：毫米

四到六位为ABC坐标值，单位：弧度

第七位保留

"posDeg":[0.0,0.0,1,2,3.3,44,5.55]： 角度点位值 直角、工具、用户坐标下四到六位ABC弧度值转为角度值，其余状态数值与pos一致

"coord":-1      坐标模式：	1-控制器当前坐标 0-关节坐标 1-直角坐标 2-工具坐标  3-用户坐标 4-电机位置

"configuration":1    姿态或(SCARA)左右手

### 运行参数设置

#### 设置时发送：

*0x2801 INTERPOLATION_MODE_SET

```json
{
"absolutePosResolution": 0.010, // 绝对位置分辨率 0.0001 - 0.1 度
"interpolationMethod": 0, // 机器人插补方式 0：S型，1：梯形，2：加加插补
"runDelayTime": 500, // 运行延时时间 500 - 20000 毫秒
"stopTime": 240 // 暂停时间 240-2000 毫秒
}
```

#### 获取时发送：

*0x2802 INTERPOLATION_MODE_INQUIRE

data: 无

#### 返回

*0x2803 INTERPOLATION_MODE_RESPOND

```json
{
"absolutePosResolution": 0.010,  //绝对位置分辨率 0.0001 - 0.1 度
"interpolationMethod": 0,    //机器人插补方式 0：s型，1：梯形 2：加加插补
"runDelayTime": 500,  //运行延时时间500 - 20000 毫秒
"stopTime": 240     //暂停时间   240-2000 毫秒
}
```

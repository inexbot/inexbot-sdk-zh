# 喷涂工艺

## 模拟量设置

模拟量设置

*0x4701  SPRAY_ANALOGGROUP_SET

{

"robot":1,

"ID":1,

"flow":1.1, //流量

"sector":2.2, //扇形

"atom":3.3, //雾化

"note":"", //注释

}

查询

*0x4702  SPRAY_ANALOGROUP_INQUIRE

{

"robot":1,

"ID":1

}

返回

*0x4703  SPRAY_ANALOGROUP_RESPOND

data：同0x4701

## 手动操作

手动操作中的选择和按钮

*0x4711   SPRAY_HAND_OPERATION_SET

{

"robot":1,

"type":1,//1:喷枪，2:时序组号，3:除尘，4:喷漆，5:清洗，6:油量测试，7:颜色切换

"value":1

}

查询

*0x4712  SPRAY_HAND_OPERATION_INQUIRE

{

"robot":1

}

返回

*0x4713  SPRAY_HAND_OPERATION_RESPOND

{

"robot":1,

"gun":1, //喷枪编号

"sequential":1, //时序组号

"dust":0, //除尘使能

"paint":0, //喷漆使能

"oilTest":0, //油量测试

"color":1 //颜色切换

}

手动操作中的模拟量设置

*0x4714   SPRAY_CUR_ANALOG_SET

{

"robot":1,

"anaID":1, //模拟量组号

"flow":1.1, //流量

"sector":2.2, //扇形

"atom":3.3, //雾化

}

查询

*0x4715   SPRAY_CUR_ANALOG_INQUIRE

{

"robot":1，

"anaID":-1 //-1：查询组号，0-99切换组号

}

返回

*0x4716   SPRAY_CUR_ANALOG_RESPOND

{

"robot":1,

"anaID":1, //模拟量组号

"flow":1.1, //流量

"sector":2.2, //扇形

"atom":3.3, //雾化

}

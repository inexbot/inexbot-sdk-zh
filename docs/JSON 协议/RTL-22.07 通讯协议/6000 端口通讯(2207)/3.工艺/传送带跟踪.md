![](https://ones.inexbot.com/wiki/api/wiki/external-editor/RnqpQ1Yp/KRNP9oDe/resources/mwhKYVmm_TDWeCfMT9vNccDMu1nkCNuhc26e7bNkdWU.png)

设置传送带参数

*0x4801 TRACK_CONVEYOR_CONVEYORPARAM_SET

```json
{
"compensation":{
"encoderVal":66, //跟踪补偿编码器值
"time":10.0      // 跟踪补偿时间
},
"conveyor":{
"encoderDirection":1,    // 1：正向；-1：反向
"encoderResolution":44.330,   // 编码器分辨率，double
"maxEncoderVal":2147483647.0,  // 编码器计数最大值，double
"minEncoderVal":-2147483648.0, // 编码器计数最小值，double
"posRecordMode":0,    //传送带位置模式 0-编码器；1-恒速设置
"speed":9.0,    //当传送带位置模式为恒速设置时候，设置传送带速度
"userCoord":4         // 用户坐标系1~9
},
"conveyorID":1,     //传送带工艺号
"robot":1,
"track":{
"height":1,   //追踪目标高度  0-传感器感知 1-跟踪指令示教
"track_on_run_mode_with_target_overrun":1   //目标超限运行方式 0-等待下一个目标 1-跳行到跟踪结束运行
}
}
```

设置参数识别

*0x4804 TRACK_CONVEYOR_POSCHECKPARAM_SET

```json
{
"conveyorID": 1, //传送带工艺号
"detectSrc": {
"type": 0, //工件检测信号源 0-视觉 1-IO 2-全局变量
"globalVar":"GB018", //当工件检测信号源为2-全局变量时候，生效
"DI_capturePos":9,  //当工件检测信号源为 1-IO时候，生效
"visionID": 3, //视觉工艺号
"vision_io_filter_type": 0, //视觉IO信号过滤 0-信号触发过滤 1-信号消失过滤
"vision_latch_encoder_value_type": 1 //视觉锁存编码器值 0-触发锁存 1-收到数据锁存
},
"identification": {
"communication": 1, //视觉通讯方式 0-以太网；1-Modbus
"sensorTrg": 1, //传感器触发方式 0-低电平触发， 1-高电平触发；
"type": 0   //工件识别方式 0-视觉；1-传感器
},
"robot": 1
}
```

标定位置

注：标记

*0x4833   TRACK_CONVE YOR_POSITION_CALIBRATION

```json
{
"robot":1,
"conveyorID":1,
"type":1   //1,2,3,4..7
}
```

type:

1-跟踪开始X点

2-跟踪范围X最大

3-跟踪范围Y最小

4-跟踪范围Y最大

5-跟踪范围Z最小

6-跟踪范围Z最大

7-最迟接收位置

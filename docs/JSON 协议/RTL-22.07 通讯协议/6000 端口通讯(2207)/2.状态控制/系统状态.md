*0x3402 VERSIONNUM_INQUIRE

- Bool 类型
- 初始化是否完成：true：完成，false：未完成

```json
{
    "version": "v1.0-rc1-67-gf34dae7",
    "servoVersion": xx,
    "ocmVersion": xx,
    "ioVersion": xx
}
```json
"version":"v1.0-rc1-67-gf34dae7",		//版本号内容

"servoVersion":xx,               //伺服版本

"ocmVersion":xx,               //静态库版本

"ioVersion":xx,                 //IO板版本

### 控制器时间查询

#### 时间查询

- 命令字：0x2304 DEADMAN_MODE_SET
- 命令字：0x5742 CONTROLLER_TIME_INQUIRE
- "type":0
- int 类型
- 1:获取系统开始时间至现在秒数，0:现实时间
- "format":0

```json
{
"type":0,
"format":0
}
```
#### 返回0x5742查询结果

- int 类型
- 1:获取北京时间，0:获取时间戳
- 命令字：0x5743 CONTROLLER_TIME_RESPOND
- "tType":0
- int 类型
- 1:获取系统开始时间至现在秒数，0:现实时间
- "format":0
- int 类型
- 1:获取北京时间，0:获取时间戳
- "date":
- format为1时用格林威治标准时间表示
- 包含日期和时间信息，分别记录年、月、日、时、分、秒、毫秒等值，通常用于表示某一时刻的完整时间信息。
- "year":2023
- int 类型
- 日期中的年份
- mon":1
- int 类型
- 日期中的月份
- "day":1
- int 类型
- 日期中的天
- "hour":1
- int 类型
- 时间中的小时
- "min":1
- int 类型
- 时间中的分钟
- "sec":
- int 类型
- 时间中的秒数
- "msec":1
- int 类型
- 时间中的毫秒数
- "timestamp":
- 以秒和纳秒的形式记录时间戳，用于高精度的时间计算，通常与日期结合使用。
- "s":1
- int 类型
- 时间戳中的秒数
- "ns":1

```json
{
  "tType": 0,
  "format": 0,
  "date": {
    "year": 2023,
    "mon": 1,
    "day": 1,
    "hour": 1,
    "min": 1,
    "sec": 1,
    "msec": 1
  },
  "timestamp": {
    "s": 1,
    "ns": 1
  }
}
```

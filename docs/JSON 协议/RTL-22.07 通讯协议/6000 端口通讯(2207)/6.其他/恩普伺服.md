# 恩普伺服

#### 碰撞检测

*0x3203 NEP_FUNCTION_SET

｛

"robot":1,

"collisionDetectionFlag":true,		//碰撞检测使能

“dragTeachFlag”：false,			//拖动示教使能

"collisionThreshold":[1300,1300,1300,1300,1300,1300]	//各轴碰撞阈值

}

#### 碰撞清除

*0x3206 NEP_FUNCTION_CLEAR

```json
{
"robot":1  //碰撞清除
}
```

#### 控制器回复：

*0x3207 NEP_FUNCTION_CLEAR_RESPOND

```json
{
"robot":1
"clear":true;   //true:碰撞清错成功 false:碰撞清错失败
}
```

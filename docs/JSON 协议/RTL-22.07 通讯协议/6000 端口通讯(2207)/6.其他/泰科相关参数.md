# 泰科相关参数

#### 设置防抱闸压紧使能：

*0x7701 BRAKE_AVOIDCRUSHED_SET

```json
{
"enable":true or false     //防抱闸压紧使能1使能，0关闭
}
```

#### 查询：

*0x7702 BRAKE_AVOIDCRUSHED_INQUIRE

data:无

#### 控制器回复：

*0x7703 BRAKE_AVOIDCRUSHED_RESPOND

```json
{
"enable":1 or 0     //1使能，0关闭
}
```

#### 设置延时参数：

*0x7704 BRAKE_AVOIDCRUSHED_DELAY_SET

```json
{
"enableDelay":0.5      //使能延时
"brakeOnDelay":0.5    //开抱闸延时
"brakeOffDelay":0.5    //抱闸关闭后延时
}
```

#### 查询：

*0x7705 BRAKE_AVOIDCRUSHED_DELAY_INQUIRE

data:无

#### 控制器回复：

*0x7706 BRAKE_AVOIDCRUSHED_DELAY_RESPOND

```json
{
"enableDelay":0.5      //使能延时
"brakeOnDelay":0.5    //开抱闸延时
"brakeOffDelay":0.5    //抱闸关闭后延时
}
```

#### 抱闸参数设置发送：

*0x7707 BRAKE_AVOIDCRUSHED_PARAM_SET

```json
{
"jointNum":1-6     //关节数
"EncodeNum":1 or 2  //编码器个数
"Encode1Resolusion":55555  //编码器1位数
"Encode2Resolusion":55555   //编码器2 分辨率
"BrakeType":1 or 2  //1 插销式 , 2 刹片式
"Distance": 111    //运动距离
"CheckDistance":111  //检测距离
"CheckTorq":111   //检测力矩
}
```

#### 查询：

*0x7708 BRAKE_AVOIDCRUSHED_PARAM_INQUIRE

{

"jointNum":1-6    //关节数

｝

#### 控制器回复：

*0x7709 BRAKE_AVOIDCRUSHED_PARAM_SRESPOND

```json
{
"jointNum":1-6     //关节数
"EncodeNum":1 or 2  //编码器个数
"Encode1Resolusion":55555  //编码器1位数
"Encode2Resolusion":55555   //编码器2 分辨率
"BrakeType":1 or 2  //1 插销式 , 2 刹片式
"Distance": 111    //运动距离
"CheckDistance":111  //检测距离
"CheckTorq":111   //检测力矩
}
```

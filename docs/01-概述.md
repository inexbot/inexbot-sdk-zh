# 概述

纳博特（NexDroid）是国内领先的工业机器人控制器厂商。纳博特自主研发的开放型 NexDroid 软件平台，提供稳定可靠、功能丰富的二次开发接口（NexDroidAPI）。NexDroid 分离了运控算法层与应用层，让合作伙伴只需关注行业需求。

利用 NexDroidAPI，结合行业经验，就可以开发专有自动化装备，如 3C 电子、医疗、核能、工程机械、风电等行业设备。

---

# 1. 平台架构

纳博特软件平台提供多个层级的二次开发能力：

| 开发层级 | 说明 | 开发语言 |
|----------|------|----------|
| **控制器二次开发** | 直接在控制器内运行程序 | Linux, C++ |
| **示教器二次开发** | 定制专属 UI 界面和功能 | Linux, Qt/C++ |
| **上位机 SDK** | 通过 API 直接控制机器人 | Windows/Linux, C++/C#/Python |
| **JSON 协议** | 基于 JSON 格式的 TCP 通讯 | 任意语言 |
| **ROS 开发** | ROS 机器人操作系统集成 | Linux, C++/Python |
| **HAL 层开发** | 硬件抽象层，访问底层硬件 | Linux, C++ |
| **EtherCAT 主站** | 工业总线主站开发 | Linux, C++ |

# 3. 主要内容

## 3.1 二次开发接口

纳博特机械臂二次开发接口将机械臂复杂的功能进行接口化，用户无需进行底层开发即可快速使用机械臂的高级控制功能。

**支持的开发环境：**
- C、C++、Qt、C#、LabVIEW、Python、JSON、ROS

**涵盖内容：**
- API 接口详细说明
- 参数解释
- 示例代码

## 3.2 Demo 示例

纳博特 Demo 示例提供了多个基于纳博特控制器典型应用的参考，包括：

- 坐标系操作
- 力控
- 抓取
- IO 功能
- 外部轴控制
- ModbusRTU 通信
- 样条曲线运动
- 角度透传
- 在线编程
- 算法等

## 3.3 二次开发资源包

软件二次开发资源包包括：
- API 接口包
- SDK 包
- ROS 包

# Ethernet/IP 通讯

# 1. 概述

Ethernet/IP（EIP）是一种基于以太网的工业通讯协议，纳博特控制器支持 EIP 从站模式。通过 JSON 协议可以配置 EIP 参数、连接参数、读写数据等。

---

# 2. 设备参数配置（0x7360 / 0x7361 → 0x7362）

## 设置 EIP 设备参数（0x7360）

| 字段 | 类型 | 说明 |
|------|------|------|
| `type` | int | 路数：1, 4, 8, 12, 16 |
| `grouptype` | int | 组号方式：0=手填，1=全局int变量，2=全局int绑定变量 |
| `group` | int | 组号数字 |
| `addtype` | int | 数值存入方式：0=全局int，1=全局int绑定，2=全局double，3=全局double绑定 |
| `add` | int | 数值存入的数字 |
| `num` | int | 编号：1-10=输出，11-20=输入 |
| `value` | - | 值 |
| `able` | bool | 使能开关 |

**示例：**

```json
{
  "type": 4,
  "grouptype": 1,
  "group": 3,
  "addtype": 2,
  "add": 100,
  "num": 5,
  "value": 1,
  "able": true
}
```

## 查询 EIP 设备参数（0x7361 → 0x7362）

无请求数据，响应格式同设置。

---

# 3. 连接参数配置（0x7363 / 0x7364 → 0x7365）

## 设置 EIP 连接参数（0x7363）

| 字段 | 类型 | 说明 |
|------|------|------|
| `enable` | bool | 连接开关 |
| `writenum` | int | 写入长度，范围[0,256] |
| `readnum` | int | 读取长度，范围[0,256] |
| `scancycle` | int | 扫描周期 |
| `outtimecycle` | int | 超时周期 |
| `networkname` | string | 网口名称 |
| `writetype` | int | 本机写类型：0=全局bool变量，1=全局bool绑定变量 |
| `writeadd` | int | 本机写的位置 |
| `readtype` | int | 本机读类型：0=全局bool变量，1=全局bool绑定变量 |
| `readadd` | int | 本机读的位置 |

**示例：**

```json
{
  "enable": true,
  "writenum": 0,
  "readnum": 0,
  "scancycle": 0,
  "outtimecycle": 0,
  "networkname": "",
  "writetype": 0,
  "writeadd": 0,
  "readtype": 0,
  "readadd": 0
}
```

## 查询 EIP 连接参数（0x7364 → 0x7365）

无请求数据，响应格式同设置。

---

# 4. 连接状态（0x7366 / 0x7367）

## 查询 EIP 连接状态（0x7366 → 0x7367）

无请求数据，响应返回连接状态。

---

# 5. 读取数据配置（0x7368 / 0x7369）

## 设置读取数据状态（0x7368）

| 字段 | 类型 | 说明 |
|------|------|------|
| `num` | int | 编号：0-9=输出，10-19=输入 |
| `value` | int | 值 |

## 设置 EIP 接收数据同步模式（0x7369）

| 字段 | 类型 | 说明 |
|------|------|------|
| `state` | bool | 同步模式：true=EIP参数随接收数据变化，false=不变 |

---

# 6. 命令字汇总

| 命令 | 功能 | 方向 |
|------|------|------|
| 0x7360 | 设置设备参数 | 请求 |
| 0x7361 | 查询设备参数 | 请求 |
| 0x7362 | 设备参数 | 响应 |
| 0x7363 | 设置连接参数 | 请求 |
| 0x7364 | 查询连接参数 | 请求 |
| 0x7365 | 连接参数 | 响应 |
| 0x7366 | 查询连接状态 | 请求 |
| 0x7367 | 连接状态 | 响应 |
| 0x7368 | 设置读取数据状态 | 请求 |
| 0x7369 | 设置同步模式 | 请求 |

---

# 7. 编号说明

| 编号范围 | 类型 |
|----------|------|
| 1-10 | 输出 |
| 11-20 | 输入 |

编号方式（grouptype）：
- 0：手填组号
- 1：全局int类型变量
- 2：全局int类型绑定变量

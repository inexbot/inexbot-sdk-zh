# 其他外设与编码器

# 1. 外部轴编码器（0x7200 / 0x7201 → 0x7202）

## 设置编码器溢值参数（0x7200）

| 字段 | 类型 | 说明 |
|------|------|------|
| `robot` | int | 机器人号，范围[1,4] |
| `externalGroupNum` | int | 轴组号，范围[1,3] |
| `externalAxis` | array | 外部轴编码器参数列表 |
| `externalAxis[].encode` | array | 编码器最大最小值参数列表 |
| `externalAxis[].encode[].max` | double | 编码器最大值，范围[-2^32, 2^32] |
| `externalAxis[].encode[].min` | double | 编码器最小值，范围[-2^32, 2^32] |

**示例：**

```json
{
  "robot": 2,
  "externalGroupNum": 1,
  "externalAxis": [
    {
      "encode": [
        { "max": "100.0", "min": "-100.0" }
      ]
    }
  ]
}
```

## 查询编码器溢值参数（0x7201 → 0x7202）

**请求：**

```json
{
  "robot": 2,
  "externalGroupNum": 1
}
```

**响应：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `robot` | int | 机器人号 |
| `externalGroupNum` | int | 轴组号 |
| `externalAxis` | array | 参数列表 |
| `externalAxis[].enable` | int | 多圈值溢出计数：0=未开启，1=已开启 |
| `externalAxis[].encode` | array | 编码器参数 |

# 3. 硬盘状态（0x7230 / 0x7231 / 0x7232）

## 查询硬盘状态（0x7230 → 0x7231）

```json
{}
```

**响应：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `Estimated Available Time` | int | 预计可用时间，单位小时 |
| `get_harddisk_warning_time` | int | 硬盘预警时间，单位小时 |

## 设置硬盘预警时间（0x7232）

```json
{
  "set_harddisk_warning_time": 50
}
```

范围：(0, 9999) 小时

# 4. 伺服内部参数（0x7240 / 0x7241 → 0x7242）

控制器分10个参数为一组发送伺服内部参数给上位机。

## 设置伺服参数（0x7240）

| 字段 | 类型 | 说明 |
|------|------|------|
| `robot` | int | 机器人号，范围[1,4] |
| `servoNum` | int | 轴号 |
| `temporary_save` | int | 存储方式：0=永久存储，1=临时存储 |
| `servo` | object | 伺服参数列表 |

**参数结构：**

```json
{
  "参数名称": {
    "value": 100.0
  }
}
```

**示例：**

```json
{
  "robot": 1,
  "servoNum": 1,
  "temporary_save": 0,
  "servo": {
    "位置指令低通滤波器截止频率": { "value": 100.0 },
    "位置环比例增益1": { "value": 1000.0 },
    "电机反电势系数": { "value": 53.0 }
  }
}
```

## 查询伺服参数（0x7241 → 0x7242）

**请求：**

```json
{
  "robot": 1,
  "servoNum": 1
}
```

**响应：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `robot` | int | 机器人号 |
| `servoNum` | int | 轴号 |
| `totalNum` | int | 参数总数 |
| `sendNum` | int | 本次发送的参数数量 |
| `servo` | object | 参数列表 |

**参数详情结构：**

```json
{
  "参数名称": {
    "hex": false,
    "index": 8299,
    "lowLimit": 1,
    "unit": "0.1HZ",
    "upLimit": 1000,
    "value": 100.0
  }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `hex` | bool | 是否为十六进制 |
| `index` | int | 参数地址 |
| `lowLimit` | int | 最小限制 |
| `upLimit` | int | 最大限制 |
| `unit` | string | 单位 |
| `value` | - | 当前值 |

# 5. 星辰一号编码器（0x7250 / 0x7251 / 0x7252 → 0x7253）

## 查询编码器范围（0x7250 → 0x7251）

**请求：**

```json
{ "encodorPort": 1 }
```

**响应：**

```json
{
  "encodorPort": 1,
  "encoderMax": 100,
  "encoderMin": 0
}
```

## 查询编码器值（0x7252 → 0x7253）

**请求：**

```json
{ "encodorPort": 1 }
```

# 通讯协议对比

| 协议 | 类型 | 典型应用 |
|------|------|---------|
| 数字IO | 点位输入输出 | 传感器、继电器、指示灯 |
| Modbus | 工业总线 | PLC、变频器、仪表 |
| CANopen | CAN总线 | 运动控制、从站设备 |
| Ethernet/IP | 工业以太网 | PLC、视觉系统 |
| OPC UA | 跨平台通讯 | SCADA、MES系统 |
| TCP | 通用通讯 | 定制化设备集成 |
| FinsTCP | PLC通讯 | 欧姆龙PLC |

# 命令字分布

| 命令字范围 | 功能 |
|------------|------|
| 0x7000-0x7022 | CANopen 总线、ENI、从站 |
| 0x7100-0x7110 | 数字IO |
| 0x7200-0x7252 | 编码器、硬盘、伺服参数 |
| 0x7300-0x7309 | Modbus 主站/从站 |
| 0x7320-0x7324 | TCP 通讯 |
| 0x7340-0x7346 | FinsTCP 通讯 |
| 0x7350-0x7352 | OPC UA |
| 0x7360-0x7369 | Ethernet/IP |

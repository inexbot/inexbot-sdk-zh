# 数据上传

# 1. 概述

数据上传功能允许控制器将采集的机器人运行数据（位置、速度、IO状态等）按指定周期上传到远程 FTP 服务器，便于数据分析和监控。

---

# 2. 数据传输开关（0x7330 / 0x7331 → 0x7332）

## 设置数据传输开关（0x7330）

```json
{
  "enable": true,
  "robot": 1
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `enable` | bool | 数据传输开关 |
| `robot` | int | 机器人号，范围[1,4] |

## 查询数据传输开关（0x7331 → 0x7332）

```json
{ "robot": 1 }
```

**响应：**

```json
{
  "enable": false,
  "robot": 1
}
```

---

# 3. 数据上传参数（0x7333 / 0x7334 → 0x7335）

## 设置数据上传参数（0x7333）

| 字段 | 类型 | 说明 |
|------|------|------|
| `serverIP` | string | FTP 服务器 IP |
| `serverPort` | int | 端口号，范围(0,65535] |
| `username` | string | 用户名 |
| `password` | string | 密码 |
| `filepath` | string | 文件路径 |
| `uploadMode` | string | 上传方式（默认 `"FTP"`） |
| `dataCollectTime` | float | 数据采集周期，单位秒，范围[0,999999] |
| `dataUploadTime` | float | 数据上传周期，单位秒，范围[0,999999] |
| `fileSuffix` | string | 文件格式：`"csv"` 或 `"txt"` |
| `sendInfoFileFlag` | bool | 是否发送说明文档 |
| `robot` | int | 机器人号，范围[1,4] |

**示例：**

```json
{
  "serverIP": "192.168.1.14",
  "serverPort": 21,
  "username": "admin",
  "password": "123456",
  "filepath": "/data",
  "uploadMode": "FTP",
  "dataCollectTime": 1.0,
  "dataUploadTime": 30.0,
  "fileSuffix": "csv",
  "sendInfoFileFlag": false,
  "robot": 1
}
```

## 查询数据上传参数（0x7334 → 0x7335）

```json
{ "robot": 1 }
```

**响应：** 同设置参数格式。

---

# 4. 数据格式配置（0x7336 / 0x7337 → 0x7338）

数据格式配置分四次进行，分别设置说明文档文件名、说明文档文件内容、数据文档文件名、数据文档文件内容。

## 设置数据格式（0x7336）

| 字段 | 类型 | 说明 |
|------|------|------|
| `robot` | int | 机器人号，范围[1,4] |
| `type` | int | 格式类型：1=说明文档文件名，2=说明文档文件内容，3=数据文档文件名，4=数据文档文件内容 |
| `format` | string | 格式字符串 |

**示例：**

```json
{
  "robot": 1,
  "type": 1,
  "format": "Robot-R1_$Y%-$m%-$d%_$H%-$M%-$S%"
}
```

此格式表示文件名：`Robot-R1_年-月-日_时-分-秒.csv`

## 查询数据格式（0x7337 → 0x7338）

```json
{
  "robot": 1,
  "type": 1
}
```

---

# 5. 格式变量说明

| 变量 | 含义 |
|------|------|
| `$Y%` | 年（4位） |
| `$m%` | 月（2位） |
| `$d%` | 日（2位） |
| `$H%` | 时（24小时，2位） |
| `$M%` | 分（2位） |
| `$S%` | 秒（2位） |
| `$ROBOT%` | 机器人号 |
| `$SEQ%` | 序号 |

---

# 6. 使用流程

1. **配置服务器参数**（0x7333）：设置 FTP 服务器地址、端口、用户名密码
2. **配置数据格式**（0x7336）：分4次设置文件和内容格式
3. **开启数据传输**（0x7330）：设置 `enable: true`

数据将按 `dataCollectTime` 周期采集，按 `dataUploadTime` 周期上传到 FTP 服务器。

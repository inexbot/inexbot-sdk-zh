# 04-工具手与用户坐标

## 工具手

### 查询工具手列表（0x3900 → 0x3901）

```json
{
  "toolList": [
    {"valid": true, "tool": 1},
    {"valid": true, "tool": 2},
    {"valid": false, "tool": 999}
  ]
}
```

### 查询当前工具手（0x3902 → 0x3903）

```json
{
  "curTool": 1
}
```

### 设置当前工具手（0x3904 → 0x3905）

```json
{
  "curTool": 1
}
```

### 查询工具手参数（0x3906 → 0x3907）

```json
{
  "angle": [0, 0, 0],
  "flange": 1,
  "load": 10.5,
  "tool": 1,
  "valid": true,
  "x": 0, "y": 0, "z": 0,
  "xyz": [0, 0, 0, 0, 0, 0]
}
```

- `tool`: 工具手编号[1,999]
- `valid`: 是否有效
- `x,y,z`: 位置偏移值（单位：mm）
- `angle`: 角度偏移值（单位：°）
- `xyz`: 完整姿态偏移[Rx,Ry,Rz]（单位：°）
- `flange`: 关联的法篮盘编号
- `load`: 负载重量（单位：kg）

### 设置工具手参数（0x3908 → 0x3909）

### 工具手TCP位置法标定（0x390D → 0x390E）

```json
{
  "point": [...],

```

### 删除工具手（0x390F → 0x3910）

```json
{
  "tool": 1
}
```

### 工具手碰撞检测参数（0x3911 → 0x3912）

```json
{
  "enable": true,
  "sensitivity": 0.5,
  "tool": 1
}
```

- `sensitivity`: 灵敏度[0,1]

## 用户坐标

### 查询用户坐标列表（0x3A00 → 0x3A01）

```json
{
  "userCoordList": [
    {"coord": 1, "valid": true},
    {"coord": 2, "valid": true},
    {"coord": 999, "valid": false}
  ]
}
```

### 查询当前用户坐标（0x3A02 → 0x3A03）

```json
{
  "curUserCoord": 1
}
```

### 设置当前用户坐标（0x3A04 → 0x3A05）

```json
{
  "curUserCoord": 1
}
```

### 查询用户坐标参数（0x3A06 → 0x3A07）

```json
{
  "angle": [0, 0, 0],
  "coord": 1,
  "name": "TestCoord",
  "point": [...],
  "valid": true,
  "x": 100, "y": 200, "z": 300,
  "xyz": [0, 0, 0, 0, 0, 0]
}
```

- `coord`: 坐标编号[1,999]
- `x,y,z`: 原点位置（单位：mm）
- `angle`: 旋转角度[Rx,Ry,Rz]（单位：°）
- `point`: 原点位置的机器人位置数据

### 用户坐标三点法标定（0x3A0A → 0x3A0B）

```json
{
  "angle": [0, 0, 0],
  "coord": 1,
  "point": [point1, point2, point3],
  "x": 100, "y": 200, "z": 300
}
```

- `point`: 三个标定点的位置数据

# 03-标定与坐标

## 1. 四点标定

### 四点标定计算（0x3700）

```json
{
  "ParamOf4Points": [85.0, 70.0]
}
```

- `ParamOf4Points`: [L1, L2] 机器人末梢从A点到C点的X/Y移动距离

### N号点标定（0x3701 → 0x3702）

```json
{
  "pointNum": 0,
  "status": 1
}
```

- `pointNum`: 标定点编号[0,3]
- `status`: 0取消标定，1标定

### 标定计算结果（0x3703）

```json
{
  "dtheta": [-1.0, -1.0],
  "length": [-1.0, -1.0],
  "result": false
}
```

- `dtheta`: 轴零点偏移列表，长度2，全为-1表示计算失败
- `length`: 杆长度列表，长度2

### 应用标定结果到DH参数（0x3704 → 0x3705）

### 查询标定信息（0x3706）

```json
{
  "dtheta": [null, null],
  "length": [null, null],
  "result": false,
  "status": [0, 1, 0, 1]
}
```

- `status`: 点位标定状态，0未标定，1已标定

## 2. 20点标定

### 标定计算请求（0x3710）

```json
{
  "calibrationType": 20,
  "noCalZero": false,
  "robot": 1,
  "tool": 1
}
```

- `calibrationType`: 2/12/15/20/21点法
- `noCalZero`: 是否校准零点
- `tool`: 工具手号[1,999]

### N号点标定（0x3711 → 0x3712）

```json
{
  "calibrationType": 20,
  "point": 1,
  "robot": 1,
  "tool": 1
}
```

### 标定计算完成结果（0x3713）

```json
{
  "distance": 464.473876425245,
  "result": true,
  "robot": 1
}
```

- `distance`: 计算结果，-1失败，>10需重新标定

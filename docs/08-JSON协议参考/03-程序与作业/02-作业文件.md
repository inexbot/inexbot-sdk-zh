# 02-作业文件

## 1. 程序运行总数清零（0x5600）

```json
{"robot": 1}
```

## 2. 运行程序（0x5601）

启动程序运行。

```json
{
  "robot": 1,
  "jobname": "TTT",
  "suffixname": ".JBR",
  "stepRunSum": 2,
  "callerName": "tttt",
  "stepRun": false,
  "index": 0,
  "noFileRun": false,
  "notJudgeSafePos": false,
  "line": 1,
  "globalStart": false,
  "continueRun": 0,
  "startOver": 0
}
```

- `robot`: 机器人号[1,4]
- `jobname/suffixname`: 程序名和后缀（.JBR前台/.JBP后台局部/.JBPG后台全局）
- `line`: 程序开始运行的行号
- `noFileRun`: 无文件运行模式
- `continueRun`: 继续运行（1继续，0从头）
- `startOver`: 重头运行（1重头，0继续）
- `globalStart`: 多机模式

## 3. 停止程序（0x5602）

```json
{"robot": 1}
```

## 4. 程序执行状态设置（0x5603）

改变程序执行模式。

```json
{
  "robot": 1,
  "type": 5
}
```

- `type`: 0单步运行，1暂停，2停止但不下电，5停止但不下电（停止优先）

注意：多机模式下暂停和停止时`robot`设置成0表示全部机器人。


- `switch`: false正序，true倒序

### 查询正序倒序（0x5611 → 0x5612）

## 7. 循环次数（0x5620/0x5621/0x5622）

### 设置循环次数（0x5620）

```json
{
  "robot": 1,
  "index": 1
}
```

- `index`: 总循环次数，0表示无限循环

### 查询循环次数（0x5621 → 0x5622）

```json
// 响应
{
  "robot": 1,
  "count": 0,
  "index": 1
}
```

- `count`: 当前已运行次数

## 8. 全局线程（0x5630/0x5631/0x5632/0x5633/0x5634）

### 设置全局线程自启动（0x5630）

```json
{
  "robot": 1,
  "name": "Q1"
}
```

- `name`: 全局后台程序名

### 查询/响应全局线程自启动（0x5631 → 0x5632）

### 全局线程启动控制（0x5633）

```json
{
  "robot": 1,
  "type": 1
}
```

- `type`: 1开始，0停止

### 查询全局后台运行状态（0x5634 → 0x50c6）

```json
// 请求
{"robot": 1, "runName": 1}
// 响应
{"robot": 1, "hasRun": [false, true]}
```

- `hasRun`: 各全局后台程序自启状态数组

## 9. 追加运动模式（0x5640/0x5641/0x5642/0x5643/0x5644/0x5645）

追加运动模式是不使用程序直接运行指令的方式。

### 打开/关闭追加运动模式（0x5640）

```json
{
  "robot": 1,
  "open": true
}
```

- `open`: true打开，false关闭

### 查询追加运动模式（0x5641 → 0x5642）

### 插入指令向量（0x5643）

```json
{
  "robot": 1,
  "isContinue": false,
  "data": [
    {cmd->toJson()},
    {cmd->toJson()}
  ]
}
```

- `isContinue`: true只接收存储pos，false接收并执行
- `data`: 指令json列表

### 追加运动模式暂停（0x5644）

```json
{"robot": 1}
```

### 追加运动模式启动（0x5645）

```json
{"robot": 1}
```

# 寻位跟踪工艺

# 1. 跟踪传感器类型

## 设置跟踪传感器类型（0x8740）

**请求参数：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `fileNum` | int | 跟踪文件号，范围[1,99] |
| `robot` | int | 机器人号，范围[1,4] |
| `sensorType` | int | 传感器类型：0-线激光，1-电弧，2-弧压 |

**示例：**

```json
{
  "fileNum": 1,
  "robot": 1,
  "sensorType": 0
}
```

## 查询跟踪传感器类型（0x8741 → 0x8742）

查询返回格式同设置命令。

# 2. 电弧通讯参数

## 设置电弧通讯参数（0x874D）

**请求参数：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `craftid` | int | 跟踪文件号，范围[1,99] |
| `robot` | int | 机器人号，范围[1,4] |
| `sampling.dataType` | int | 采样数据类型：0-电流，1-电压 |
| `sampling.period` | int | 采样周期，单位 ms，范围[0,1000] |

**示例：**

```json
{
  "craftid": 1,
  "robot": 1,
  "sampling": {
    "dataType": 0,
    "period": 20
  }
}
```

# 3. 电弧补偿参数

## 设置电弧跟踪左右补偿（0x8750）

**请求参数：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `craftid` | int | 跟踪文件号 |
| `robot` | int | 机器人号 |
| `lateralCompensation.accFactor` | double | 补偿加速度倍数，范围[0.1, 10] |
| `lateralCompensation.algorithmType` | int | 偏差提取类型：0-均值，1-手填值 |
| `lateralCompensation.beginCycleNum` | int | 开始采样周期数，范围[1, 100] |
| `lateralCompensation.differential` | double | 微分系数，范围[0, 1] |
| `lateralCompensation.integral` | double | 积分系数，范围[0, 1] |
| `lateralCompensation.maxSingleLen` | double | 每次最大补偿量，单位 mm，范围[0, 10] |
| `lateralCompensation.scale` | double | 比例系数，范围[0, 1] |
| `lateralCompensation.switchon` | bool | 补偿开关 |

**示例：**

```json
{
  "craftid": 1,
  "lateralCompensation": {
    "accFactor": 1.0,
    "algorithmType": 0,
    "beginCycleNum": 4,
    "differential": 0.0,
    "integral": 0.00050,
    "maxSingleLen": 2.0,
    "scale": 0.050,
    "switchon": true
  },
  "robot": 1
}
```

## 查询电弧左右补偿（0x8751 → 0x8752）

返回格式同设置命令。

# 4. 电弧跟踪路径数据记录

## 设置路径数据记录参数（0x8756）

| 字段 | 类型 | 说明 |
|------|------|------|
| `craftid` | int | 跟踪文件号 |
| `robot` | int | 机器人号 |
| `track_path_record.cycle` | double | 记录周期，范围[1, 1000000000] |

```json
  }
}
```

返回格式同设置命令。

# 5. PID 参数说明

电弧跟踪补偿采用 PID 控制：

| 参数 | 说明 | 典型范围 |
|------|------|----------|
| `scale` | 比例系数 | 0.01-0.1 |
| `integral` | 积分系数 | 0.0001-0.001 |
| `differential` | 微分系数 | 0-0.5 |
| `maxSingleLen` | 输出限幅 | 1-5 mm |
| `accFactor` | 加速度倍率 | 1.0-2.0 |

# 04-焊接工艺

## 1. 焊接装备（0x8300/0x8301/0x8302）

### 设置焊接装备（0x8300 → 0x8301 → 0x8302）

```json
{
  "basic_functions": {
    "arc_detection_time": 1.1,
    "arc_depletion_detection_time": 1.1,
    "arc_detection_ensure_time": 1.1,
    "pre_give_gas_time": 1.1,
    "close_time": 1.1,
    "weld_interruption_detection": true,
    "arc_fly": true,
    "air_close_mode": 0
  },
  "micro_adjustment": {
    "single_adjustment_current": 1.1,
    "single_adjustment_voltage": 1.1
  },
  "restart": {
    "restart_times": 1,
    "restart_distance": 1.1,
    "restart_speed": 1.1,
    "re_arc_times": 1,
    "restart_enable": true,
    "re_arc_enable": true,
    "restart_mode": 1
  },
  "antiCollision": {
    "enable": false,
    "inputPort": 0,
    "inputValue": 1,
    "stopTime": 60,
    "outputPort": 0,
    "outputValue": 1,
    "shieldFlag": false,
    "shieldTime": 30
  },
  "other": {
    "break_back_time": 1.1,
    "weld_finish_time": 1.1,
    "break_back_enable": true,
    "weld_finish_function": true,
    "simulation_set_zero": true
  }
}
```

**基础功能参数：**
- `arc_detection_time`: 电弧检测时间[0,5]s
- `arc_depletion_detection_time`: 电弧耗尽检测时间[0,5]s
- `arc_detection_ensure_time`: 电弧检测确认时间[0,5]s
- `pre_give_gas_time`: 提前送气时间[0,5]s
- `close_time`: 关气时间[-5,5]s（负数提前，正数延后）
- `weld_interruption_detection`: 焊接中断检测
- `arc_fly`: 飞行起弧
- `air_close_mode`: 关气方式（0延迟，1提前）

**微调整参数：**
- `single_adjustment_current`: 电流单次调整量[0,1000]A
- `single_adjustment_voltage`: 电压单次调整量[-1000,1000]V

**再启动/再起弧参数：**
- `restart_times`: 再启动次数[0,1000]
- `restart_distance`: 再启动距离[0,100]mm
- `restart_speed`: 再启动速度[0,100]mm/s
- `re_arc_times`: 再起弧次数[1,1000]
- `restart_enable/re_arc_enable`: 使能开关

**防碰撞参数：**
- `enable`: 防碰撞使能
- `inputPort/inputValue`: 防碰撞IO端口和触发电平
- `stopTime`: 快速停止时间[50,200]s
- `shieldFlag/shieldTime`: 屏蔽开关和时间

```json
{
  "type": 0,
```

```json
{
  "start": true,
```

```json
{
  "robot": 1,
  "type": 1,
  "value": 180
}
```

- `type`: 1设置电流，2设置电压

## 3. 焊接参数（0x8320/0x8321/0x8322）

### 设置焊接参数（0x8320）

```json
{
  "num": 1,
  "notes": "示例",
  "arcoff_parameters": {
    "arcoff_use_param": true,
    "arcoff_voltage": 20.0,
    "arcoff_current": 160,
    "arcoff_time": 0.5,
    "ramp_time": 500,
    "ramp_enable": true,
    "ramp_mode": 0
  },
  "weld_parameters": {
    "weld_current": 180,
    "weld_voltage": 20.0,
    "weld_velocity": 10.0
  },
  "arcon_parameters": {
    "arcon_use_param": true,
    "arcon_current": 190,
    "arcon_voltage": 22.0,
    "arcon_time": 0.3,
    "arcon_ramp_time": 300,
    "arcon_ramp_distance": 3.0,
    "arcon_ramp_enable": true,
    "arcon_ramp_mode": 0
  },
  "anti_adhesion_parameters": {
    "anti_adhesion_enable": true,
    "anti_adhesion_times": 2,
    "anti_adhesion_v": 18.0,
    "anti_adhesion_i": 100,
    "anti_adhesion_time": 0.2
  }
}
```

**收弧参数(arcoff_parameters)：**
- `arcoff_use_param`: 是否使用收弧参数
- `arcoff_voltage`: 收弧电压[-1000,1000]V
- `arcoff_current`: 收弧电流[0,1000]A
- `arcoff_time`: 收弧时间[0,5]s
- `ramp_time/ramp_enable/ramp_mode`: 渐变参数

**焊接参数(weld_parameters)：**
- `weld_current`: 焊接电流[0,1000]A
- `weld_voltage`: 焊接电压[-1000,1000]V
- `weld_velocity`: 焊接速度mm/s

**起弧参数(arcon_parameters)：**
- `arcon_use_param`: 是否使用起弧参数
- `arcon_current/voltage/time`: 起弧电流/电压/时间
- `arcon_ramp_*`: 渐变参数

**防粘丝参数(anti_adhesion_parameters)：**
- `anti_adhesion_enable`: 防粘丝使能
- `anti_adhesion_times/i/v/time`: 粘丝次数/电流/电压/时间

## 4. 摆焊参数（0x8330/0x8331/0x8332）

支持七种摆动方式：正弦摆、Z字摆、圆形摆、外部轴定点摆、L型摆、三角摆、8字摆

### 设置摆焊参数（0x8330）

```json
{
  "num": 1,
  "note": "示例",
  "parameter": {
    "weaveType": 0,
    "freq": 1.0,
    "amplitude": 5.0,
    "startDir": 0,
    "horizOffset": 0,
    "vertOffset": 0
  }
}
```

**摆焊参数(parameter)：**
- `weaveType`: 摆动类型（0正弦，1Z字，2圆形，3外部轴定点，4L型，5三角，6 8字）
- `freq`: 摆动频率Hz
- `amplitude`: 摆动幅度mm
- `startDir`: 起始方向
- `horizOffset/vertOffset`: 水平/竖直偏角

## 5. 焊接IO端口（0x8340/0x8341/0x8342）

### 设置焊接IO（0x8340）

```json
{
  "robot": 1,
  "weldIO": {
    "arconOutput": {"port": 1, "value": 1},
    "arcoffOutput": {"port": 2, "value": 1},
    "gasOutput": {"port": 3, "value": 1},
    "wireOutput": {"port": 4, "value": 1},
    "coolOutput": {"port": 5, "value": 1}
  }
}
```

- `arconOutput/arcoffOutput`: 起弧/收弧输出
- `gasOutput`: 气体输出
- `wireOutput`: 送丝输出
- `coolOutput`: 冷却输出

## 6. 多层多道焊接（0x8350/0x8351/0x8352）

### 设置多层多道（0x8350）

```json
{
  "num": 1,
  "robot": 1,
  "layerParam": {
    "totalLayer": 3,
    "totalPass": 2,
    "layer": [
      {"pass": 1, "weaveNum": 1, "weaveAngle": 0, "sideAngle": 0}
    ]
  }
}
```

- `totalLayer/totalPass`: 总层数/总道数
- `layer[].pass/weaveNum/weaveAngle/sideAngle`: 每层参数

## 7. 相贯线焊接（0x8360/0x8361/0x8362）

### 设置相贯线参数（0x8360）

```json
{
  "robot": 1,
  "mode": 0,
  "circleAngle": 45.0,
  "pipeRadius": 50.0,
  "weepAngle": 30.0
}
```

- `mode`: 相贯线模式
- `circleAngle`: 相贯角度°
- `pipeRadius`: 支管半径mm
- `weepAngle`: 泪滴角°

## 8. 焊接控制命令（0x8000/0x8001）

### 焊接控制（0x8000 → 0x8001）

```json
{
  "robot": 1,
  "cmd": 1,
  "IO": {"port": 1, "value": 1}
}
```

- `cmd`: 0停止，1起弧，2收弧，3点动送丝，4点动退丝
- `IO`: 同步输出的IO信号

# 视觉工艺

# 1. 试拍照

触发相机拍照，获取工件位置信息。

## 发送试拍照命令（0x8125）

**请求参数：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `cali_camera_angle` | bool | 是否记录工件旋转角度：true-记录，false-不记录 |
| `robot` | int | 机器人号，范围[1,4] |
| `visionNum` | int | 视觉工艺号，范围[0,98] |

**示例：**

```json
{
  "cali_camera_angle": false,
  "robot": 1,
  "visionNum": 0
}
```

## 接收拍照结果（0x8126）

**返回参数：**

| 字段 | 类型 | 说明 |
|------|------|------|
| `cameraPoint` | double[6] | 相机坐标点（X, Y, Z, Rx, Ry, Rz） |
| `protocol` | object | 数据解析协议配置 |
| `protocol.type` | int | 数据解析类型：0-2维，1-2维+高度，2-3维 |
| `camera_angle` | double | 工件旋转角度，范围[-180°, 180°] |
| `cameraData` | string | 接收的原始数据 |

**示例：**

```json
{
  "cameraPoint": [0, 0, 0, 0, 0, 0],
  "protocol": {
    "type": 0
  },
  "camera_angle": 0,
  "cameraData": ""
}
```

# 2. 视觉工艺说明

## 视觉工艺类型

| 类型值 | 说明 |
|--------|------|
| 0 | 2维视觉 |
| 1 | 2维+高度视觉 |
| 2 | 3维视觉 |

## 坐标系说明

相机坐标点 `cameraPoint` 为 6 轴数据：
- 前 3 个：位置 X, Y, Z（单位 mm）
- 后 3 个：姿态 Rx, Ry, Rz（单位 °）

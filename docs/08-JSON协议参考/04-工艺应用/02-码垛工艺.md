# 02-码垛工艺

## 1. 抓手参数（0x8000/0x8001/0x8002）

抓手设置定义码垛过程中用到的工具手。

### 设置抓手参数（0x8000）

```json
{
  "robot": 1,
  "craftID": 1,
  "pallet": {
    "gripperNum": 2,
    "gripper": [1, 3, 4, 1]
  }
}
```

- `robot`: 机器人号[1,4]
- `craftID`: 工艺号
- `gripperNum`: 抓手数目[1,4]
- `gripper`: 抓手工具号列表，第i个数代表抓手i的工具号

### 查询抓手参数（0x8001 → 0x8002）

```json
// 请求
{"robot": 1, "craftID": 1}
// 响应
{
  "robot": 1,
  "craftID": 1,
  "pallet": {"gripperNum": 2, "gripper": [1, 3, 4, 1]}
}
```

## 2. 托盘参数（0x8010/0x8011/0x8012）

### 设置托盘参数（0x8010）

```json
{
  "robot": 1,
  "craftID": 1,
  "pallet": {"userNum": 24}
}
```

- `userNum`: 用户坐标系编号

### 设置码垛托盘范围（0x8013）

```json
{
  "robot": 1,
  "craftID": 1,
  "pallet": {
    "tray_preview": true,
    "tray_length": 1000.0,
    "tray_width": 800.0
  }
}
```

- `tray_preview`: 是否打开托盘预览
- `tray_length`: 托盘长(1,50000)mm
- `tray_width`: 托盘宽(1,50000)mm

### 查询托盘范围（0x8014 → 0x8015）

## 3. 位置参数（0x8020/0x8021/0x8022）

### 设置位置参数（0x8020）

```json
{
  "robot": 1,
  "craftID": 1,
  "pallet": {
    "floorNum": 1,
    "enterPos": [9999.0,9999.0,9999.0,9999.0,9999.0,9999.0,9999.0],
    "realPos": [9999.0,9999.0,9999.0,9999.0,9999.0,9999.0,9999.0],
    "shiftPos": [9999.0,9999.0,9999.0,9999.0,9999.0,9999.0,9999.0]
  }
}
```

- `floorNum`: 标记原点所在层
- `enterPos`: 入口点[7]double
- `realPos`: 工件点[7]double
- `shiftPos`: 辅助点[7]double

### 查询位置参数（0x8021 → 0x8022）

## 4. 工件参数（0x8030/0x8031/0x8032）

### 设置工件参数（0x8030）

```json
{
  "robot": 1,
  "craftID": 1,
  "pallet": {
    "workpieceGapX": 0.0,
    "workpieceGapY": 0.0,
    "workpieceHeight": 50.0,
    "workpieceLength": [100.0],
    "workpieceWidth": [80.0]
  }
}
```

- `workpieceGapX`: 托盘X方向间隙[0,9999]mm
- `workpieceGapY`: 托盘Y方向间隙[0,9999]mm
- `workpieceHeight`: 工件高度(托盘Z方向)[1,9999]mm
- `workpieceLength`: 工件长度(托盘Y方向)[1,9999]mm
- `workpieceWidth`: 工件宽度(托盘X方向)[1,9999]mm

### 查询工件参数（0x8031 → 0x8032）

## 5. 垛型参数（0x8040/0x8041/0x8042）

### 设置垛型参数（0x8040）

```json
{
  "robot": 1,
  "craftID": 1,
  "pallet": {
    "stackType": 0,
    "layerCfg": [
      {"row": 3, "col": 4, "height": 1},
      {"row": 3, "col": 4, "height": 2}
    ]
  }
}
```

- `stackType`: 垛型类型（0交错，1顺序，2金字塔）
- `layerCfg[].row/col/height`: 每层行数/列数/层高序号

### 查询垛型参数（0x8041 → 0x8042）

## 6. 码垛控制（0x8050/0x8051）

### 码垛控制命令（0x8050 → 0x8051）

```json
{
  "robot": 1,
  "craftID": 1,
  "cmd": 1,
  "param": {}
}
```

- `cmd`: 0停止，1开始，2暂停，3继续，4复位
- `param`: 命令参数

## 7. 位置数据（0x8060/0x8061/0x8062）

### 计算放置位置（0x8060 → 0x8061）

```json
// 请求
{
  "robot": 1,
  "craftID": 1,
  "index": 1
}
// 响应
{
  "robot": 1,
  "craftID": 1,
  "pos": [...],
  "index": 1
}
```

- `index`: 工件序号（从1开始）
- `pos`: 计算得到的放置位置[7]double

### 查询码垛状态（0x8062）

```json
{
  "robot": 1,
  "craftID": 1,
  "curLayer": 2,
  "curRow": 1,
  "curCol": 3,
  "totalCount": 24,
  "curCount": 7
}
```

- `curLayer/curRow/curCol`: 当前层/行/列
- `totalCount/curCount`: 总数量/当前完成数

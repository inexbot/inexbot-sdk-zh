# 指令-运算类

### 加

- 说明：

- 参数1与参数2相加的值存入结果变量

- 加

- 指令名：ADD

- "desType1":2

- 变量1的类型；此处只能填2

- "desType2":2

- 变量2的类型；此处只能填2

- "logout":false

- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行

- "paraSourceGroup":{"data":0.0,"secondvalue":0,"value":1,"varname":"I001"}

- 结果变量；int类型变量或double类型变量，全局或局部变量

- "paraValueGroup1":{"data":2.0,"secondvalue":0,"value":0,"varname":""}

- 变量1；数值或int类型变量或double类型变量，全局或局部变量

- "paraValueGroup2":{"data":3.0,"secondvalue":0,"value":0,"varname":""}

- 变量2；数值或int类型变量或double类型变量，全局或局部变量

- "type":43

- ADD在枚举数列type中为43；int类型

- "userParamInt":0

- 二次开发中客户自定义，目前无意义

- "userParamString":""

```
{
  "desType1":2,

  "paraValueGroup2":{"data":5.0,"secondvalue":0,"value":0,"varname":""},
```

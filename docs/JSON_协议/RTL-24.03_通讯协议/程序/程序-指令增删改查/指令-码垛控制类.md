# 指令-码垛控制类

### 码垛开始

- 说明：

- 码垛开始的判断
- 可通过修改变量控制码第几层第几个工件

- 码垛开始

- 指令名：PALON

- "ID":1

- 码垛工艺号；int类型；取值范围[1,99]

- "STACKTYPE":0

- 码垛类型；int类型；取值范围[0,1]
- 0：码垛；1：卸垛

- "_pCurrentLayersNum":{"data":0.0,"secondvalue":1,"value":4,"varname":"GI[GI001]"}

- 当前层码垛数；int类型变量或绑定变量，全局或局部
- 当该变量的值表示string类型[-]时：表示不使用

- "_pLayersNum":{"data":0.0,"secondvalue":1,"value":1,"varname":"I[I001]"}

- 当前码垛层；int类型变量或绑定变量，全局或局部
- 当该变量的值表示string类型[-]时：表示不使用

- "_pWorkpieceSum":{"data":0.0,"secondvalue":0,"value":1,"varname":"[-]"}

- 当前已码总数；int类型变量或绑定变量，全局或局部
- 当该变量的值表示string类型[-]时：表示不使用

- "_stackSequence":1

- 单层码垛/卸垛顺序；int类型；取值范围[0,1]
- 0：正序；1：倒序

- "logout":false

- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行

- "type":70

- PALON在枚举数列type中为70

- "userParamInt":0

- 二次开发中客户自定义，目前无意义

- "userParamString":""

```
{
  "ID":1,

  "_pCurrentLayersNum":{"data":0.0,"secondvalue":1,"value":4,"varname":"GI[GI001]"},
```

- 当运动方式为关节运动时："m_vUnit"的值为2，此时速度单位为%；取值范围[1,100]

- 位移轴；int类型；取值范围[0,2]

### 码垛辅助点

- 当入口点高度比固定点低时，入口点在高度上会和固定点、辅助点处于同一值线上（侧视同一直线，俯视不是同一直线，XY轴不变）
- 当入口点高度位于固定点与辅助点之间时，入口点高度不变

- 将优化到与固定点水平的高度当入口点、辅助点高度都高于固定点时，入口点高度将优化到与辅助点水平高度

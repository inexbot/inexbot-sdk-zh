# 指令-位置变量类

### 用户坐标修改

- 说明：

- 修改或者读取用户坐标系某一轴的数值

- 用户坐标修改

- 指令名：USERFRAME_SET

- "ID":888

- 用户坐标编号；int类型；取值范围[1,999]

- "coord":"UCUSTOM(UX、UY、UZ)"

- 用户坐标参数；string类型；取值UX、UY、UZ、UA、UB、UC、UCUSTOM(UX、UY、UZ)
- UX、UY、UZ、UA、UB、UC：读取用户坐标系某一轴的数值；UCUSTOM(UX、UY、UZ)：自定义，括号中为自定义选中的用户坐标

- "logout":false

- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行

- "m_varName":"I001"

- 变量；int或double类型变量，全局或局部，写入时可手填int类型；取值范围[-2000,2000]
- 在写入时：直接手填数值修改坐标轴的数值，变量赋值的形式修改坐标轴的数值
- 在读取时：用变量的形式将选择的坐标轴的数值读取出来

- "mode":1

- 修改方式；int类型；取值范围[0,1]
- 0：读取用户坐标某一轴的值，1：修改用户坐标某一轴的值

- "type":97

- 用户坐标修改在枚举数列type中为97

- "userParamInt":0

- 二次开发中客户自定义，目前无意义

- "userParamString":""

- 二次开发中客户自定义，目前无意义

- "varType":1

```
{
  "ID":888,
  "coord":"UCUSTOM(UX、UY、UZ)",
  "logout":false,
  "m_varName":"I001",
  "mode":1,
  "type":97,
  "userParamInt":0,
  "userParamString":"",
  "varType":1
}
```

### 读取点位

- 变量值类型；int类型；取值范围[0,4]
- 0：手填值，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE

- 说明：

- 将位置变量的某个轴的值读入到浮点型变量

- 读取点位

- 指令名：READPOS

- "axis":1

- 位置变量轴；int类型；取值范围[1,13]
- 取值1~7:1~7轴，取值8~13：外部轴O1~O6；当前配置有什么轴，可以选什么轴

- "coord":"RF"

- 位置变量坐标系；string类型
- RF：关节坐标，BF：直角坐标，TF：工具坐标，UF：用户坐标

- "logout":false

- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行

- "posidname":"GE$GI001"

- 位置变量名；string类型
- 当字符串为空时表示当前:读取的是机器人当前位置的坐标
- P,GP,E,GE:将点位存入变量，读取时只需要选择想要读取的目标变量即可

- "posidtype":1

- 位置变量名标识；int类型；取值范围[0,1]
- 0：位置变量名固定，例：P001，G001；1：位置变量名不固定，例：P$I001，P$GI001，G$I001，G$GI001

- "type":92

- 读取点位在枚举数列type中为92

- "userParamInt":0

- 二次开发中客户自定义，目前无意义

- "userParamString":""

- 二次开发中客户自定义，目前无意义

- "varname":"D001"

```
{
  "axis":1,
  "coord":"RF",
  "logout":false,
  "posidname":"GE$GI001",
  "posidtype":1,
  "type":92,
  "userParamInt":0,
  "userParamString":"",
  "varname":"D001"
}
```
### 点位加

- 变量名；string类型；只能以D或GD开头，代表double类型
- 将读取到的值存入变量

- 说明：

- 在不同坐标系下给位置变量的某一个变量轴增加数值
- 修改目标点位时注意超限问题和目标位置不可达，坐标无法转换问题
- 关节超限：例如机器人一轴的正限位是180°,目标点位一轴的坐标是150°，在设置参数是给变量轴一轴赋值50°,运行指令时就会出现关节超限问题
- 坐标无法转换问题：例如目标点位是关节点位,在设置参数时选择的变量坐标系是直角坐标系，给目标点位的位置变量轴赋值，执行点位加指令，点位在运算过程中可能会出现坐标无法转换问题

- 点位加

- 指令名：POSADD

- "axis":5

- 位置变量轴；int类型；取值范围[1,13]
- 取值1~7:1~7轴，取值8~13：外部轴O1~O6；当前配置有什么轴，可以选什么轴

- "caltype":0

- 运算类型；int类型；取值范围[0,2]
- 0：ADDTYPE点位加；1：SUBTYPE点位减；2：SETTYPE点位改

- "coord":"RF"

- 位置变量坐标系；string类型
- RF：关节坐标，BF：直角坐标，TF：工具坐标，UF：用户坐标

- "datatype":1

- 变量值类型；int类型；取值范围[0,4]
- 0：手填值，1：INT，2：DOUBLE，3：GINT，4：GDOUBLE

- "desData":"GI001"

- 位置变量；string类型，手填值或int类型，double类型变量，全局或局部；取值范围[-50000，50000]

- "logout":false

- 是否被注释；bool类型
- true：在作业文件中将不执行该指令；false：未被注释，可执行

- "posidtype":0

- 位置变量名标识；int类型；取值范围[0,1]
- 0：位置变量名固定，例：P001，G001；1：位置变量名不固定，例：P$I001，P$GI001，G$I001，G$GI001

- "positionId":"GP0001"

- 需要修改点位的位置变量；string类型，P或GP或E或GE

- "type":93

- 点位加在枚举数列type中为93

- "userParamInt":0

- 二次开发中客户自定义，目前无意义

- "userParamString":""

### 复制点位

- 二次开发中客户自定义，目前无意义

- 说明：

- 将一个位置变量的点位数据复制到另一个位置变量
- 最大支持连接5个外部轴，在复制外部轴点位时只支持O1、O2、O3、O4、O5轴的复制，如果在设置参数时勾选了O6、O7轴，在执行复制点位时O6、O7轴不生效

- 复制点位

- 指令名：COPYPOS

- "cpaxles":[2,4,6]

- 选择需要复制的坐标轴数据；int类型数组
- 取值：可以有轴数个元素，每个元素代表相应轴号

- "cpaxletype":1

- 表示是否所有坐标轴数据都复制；int类型；取值范围[0,1]
- 0：表示所有坐标轴数据都复制，此时没有"cpaxles"节点；1：看"cpaxles"节点选择了什么

- "cpdposname":"GE$GI001"

- 目标位置变量名；string类型，P或E，局部或全局

- "cpdpostype":1

- 位置变量名标识；int类型；取值范围[0,1]
- 0：位置变量名固定，例：P001，G001；1：位置变量名不固定，例：P$I001，P$GI001，G$I001，G$GI001

- "cpposname":"P$GI001"

- 源位置变量名；string类型

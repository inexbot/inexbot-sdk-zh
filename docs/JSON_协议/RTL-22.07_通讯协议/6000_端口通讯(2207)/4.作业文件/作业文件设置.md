# 作业文件设置

### 作业文件执行状态设置

#### 作业文件执行模式有：

—step：0

—stop：1

—restart：2

#### 示教器需要改变当前控制器作业文件执行模式的时候，发送下面命令：

- 命令字：0x2401 JOBEXE_MODE_SET

- "type":0

- int 类型
- 作业文件执行状态：0：step，1：stop

- "robot": 1

```
{
"type":0,
"robot":1
}
```
#### 上位机使用无文件运行开关的时候，发送下面命令：

- int 类型
- 选择机器人 1
- 选择机器人的取值范围 [1，4]

- 命令字：0x2402 RUNJOB_NOFILE

- "robot": 1

```
{
"robot":1
}
```
#### 上位机使用反向运行开关的时候，发送下面命令：

- int 类型
- 选择机器人 1
- 选择机器人的取值范围 [1，4]

- 命令字：0x2403 REVERSEORDER_RUN

- "switch":true

```
{
"switch":true
}
```
#### 查询正序倒序：

- Bool 类型
- 是否反向顺序运⾏

- 命令字：0x2405 REVERSEORDER_RUN_INQUIRE

```
{
}
```
#### 反向当前正序还是倒序：

- "date": 无

- 命令字：0x2406 REVERSEORDER_RUN_RESPOND

- “switch”:false

```
{

```
### 开始运行作业文件

#### 示教器给控制器发送开始运行作业文件的时候，发送下面命令：

- Bool 类型
- false：正序   true：倒序

- 命令字：0x2501 	JOBSEND_DONE

- "robot": 1

- int 类型
- 选择机器人 1
- 选择机器人的取值范围 [1，4]

- “jobname”: "aaa"

- string 类型
- 作业文件名

- “line”:1

- int 类型
- 作业文件指令行数,不能为零，不能超过总行数

- "globalStart":false

- Bool 类型
- 是否为多机模式

- "continueRun":0

- int 类型
- 1:继续运行,0:不继续运行

- "startOver":0

- int 类型
- 1:重头运行，0:继续运行

- "continueRun"和"startOver"的组合使用

```
{
"robot":1,
"jobname":"aaa",
"line":1,
"globalStart":false,
"continueRun":0,
"startOver":0
}
```
​
​

#### 停止正在运行的作业文件

- "continueRun":1 startOver":false 断点执行
- "continueRun":0 "startOver":true 从头运行
- "continueRun":0 "startOver":false 当前行运行

- 命令字：0x2503 STOP_JOB_RUN

- "robot": 1

```
{
"robot":1
}
```

### 全局后台自启动

#### 全局后台设置自启动

*0x50c1  GLOBAL_PTHREAD_SELFSTART_SET

{

“robot”:1    //机器人号

“name”:"Q1"  //全局后台名字

}

#### 获取

*0x50c2  GLOBAL_PTHREAD_SELFSTART_INQUIRE

{

“robot”:1   //机器人号

}

#### 控制器回复

*0x50c3  GLOBAL_PTHREAD_SELFSTART_RESPOND

{

“robot”:1    //机器人号

“name”:"Q1"  //全局后台名字

}

#### 开始全局后台自启

*0x50c4  GLOBAL_PTHREAD_RUN_CONTROL

{

“robot”:1     //机器人号

"type":1      //1：开始  0 表示停止

}

#### 查询全局后台自启状态

*0x50c5  GLOBAL_PTHREAD_RUNSTATUS_INQUIRE

{

“robot”:1    //当前机器人号

“hasRun":true  //自启状态true表示启动false表示停止

"runName":1   //1：获取全局后台运行状态

}

#### 控制器回复

*0x50c6  GLOBAL_PTHREAD_RUNSTATUS_RESPOND

{

“robot”:1    //当前机器人号

“hasRun":false,true  //自启状态true表示启动false表示停止

}

- 命令字：0x3a05

- "openedJobName":"abs"

  "openedJobName":"asd"
}

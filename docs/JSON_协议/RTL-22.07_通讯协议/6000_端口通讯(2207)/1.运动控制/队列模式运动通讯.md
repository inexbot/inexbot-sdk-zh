# 队列模式运动通讯

### socket控制追加模式运动

#### 开启/关闭socket直接控制运动模式     开启后会进去特殊的运行模式，关闭后需要手动切回示教模式

*0x50B1  DIRECTMOTION_MODE_SET

{

"robot":1,

"open":true

}

#### 控制器返回开启成功/失败

*0x50B3  DIRECTMOTION_MODE_RESPOND

{

"robot":1,

"open":true

}

#### 发送作业文件指令队列

*0x50B4  DIRECTMOTION_INSERT_INSTRVEC

{

"data":

[

{ cmd->toJson() },   //指令json列表

{ cmd->toJson() },

{ cmd->toJson() }

],

"robot": 1

}

#### 暂停追加运行

*0x50B7  DIRECTMOTION_MODE_SUSPEND

{

"robot":1 //机器人1，2，3，4

}

#### 开始追加运行（暂停后，发送使用）

*0x50B8  DIRECTMOTION_MODE_START

{

"robot":1 //机器人1，2，3，4

}

#### 停止追加运行

*0x50B9  DIRECTMOTION_MODE_STOP

{

"robot":1 //机器人1，2，3，4

}

#### 设置队列模式停止不下电

*0x50ba

{

robot:1 //机器人1，2，3，4

}

# 远程程序设置

### 远程模式连接状态

#### 查询Modbus,外部Io连接状态时，发送：

- 命令字：0x5032 REMOTE_CONNECT_INQUIRE

```
{
}
```
#### 收到查询连接状态时，返回：

- “date”: 无

- 命令字：0x5033 REMOTE_CONNECT_RESPOND

- "ModbusConnect":false

- Bool 类型
- modbus连接状态，:false：未连接，true：已连接

- "ExternIOConnect":true

- Bool 类型
- 外部IO连接状态，:false：未连接，true：已连接

- "type":2

```
{
"ModbusConnect":false,
"ExternIOConnect":true,
"type":2
}
```
### 远程作业文件设置

#### 设置远程作业文件：

*0x2C01 REMOTE_JOBFILE_SET

{

"robot"：1, //机器人号

"jobFile"

[

{"name": "Q1",  "times": 1},

{"name": "Q2",  "times": 1},

{"name": "Q3",  "times": 1},

{"name": "Q4",  "times": 1},

{"name": "Q5",  "times": 1},

……                     //10个

] //name作业文件名称，times运行次数

}

#### 获取远程作业文件：

*0x2C02 REMOTE_JOBFILE_INQUIRE

{

"robot":1

}

#### 控制器收到示教盒查询作业文件名字时，返回：

*0x2C03 REMOTE_JOBFILE_RESPOND

{

"robot"：1 //机器人号

"jobFile":

[

{name:"Q1",times:3},

{name:"Q2",times:3},

{name:"",times:1},

{name:"Q4",times:2},

{name:"",times:1}

] //作业文件名称,运行次数

}

### 远程模式参数设置

#### 远程模式参数设置

*0x2C04 REMOTE_PARA_SET

{​
​"remoteDefaultSpeed":15  //远程模式速度1~100,

"reserveIsStart":1 //预约并启动 1启动 0关闭,

"reserveTrgTime":200 //确认启动时间200~1000ms,

"robot":1,

"waitingTime":500 //IO重复触发时间

}

#### 远程模式参数查询

*0x2C05 REMOTE_PARA_INQUIRE

{

"robot":1,

}

#### 返回0x2C05查询结果

*0x2C06 REMOTE_PARA_RESPOND

{​
​"remoteDefaultSpeed":15  //远程模式速度1~100,

"reserveIsStart":1 //预约并启动 1启动 0关闭,

"reserveTrgTime":200 //确认启动时间200~1000ms,

"robot":1,

"waitingTime":500 //IO重复触发时间

}

- int 类型
- 0:NAN,  1:RTU,  2:TCP

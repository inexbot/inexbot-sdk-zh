# 上位机

libnrc_host 是一个 C++ 库，提供了一套可以快速集成到项目中的网络接口（基于 Socket_API 通讯协议封装），让开发者无需关注底层协议细节，提高开发效率。生成的 API 文档 提供了所有功能的概述。

## 兼容性说明

支持 Linux、Windows 平台，以及多种高级开发语言，方便开发者选择最适合自己的平台进行开发。

| 类别 | 兼容范围 |
| 系统 | 支持 Windows\Linux，x86、x86_64、arm、arm64 架构 |
| 开发环境 | 针对自己的开发语言使用对应的 QT,Python,C# 推荐的 IDE，如 Qtcreator, VS Studio，VSCode 等. |

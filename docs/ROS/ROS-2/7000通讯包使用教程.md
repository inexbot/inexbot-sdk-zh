# 7000通讯包使用教程

目前，我们支持ROS2与inexbot控制器之间基于socket协议，通过7000端口进行通讯，功能包在相关下载中，本节将介绍该通讯包的使用流程

##### 1.修改控制器配置

ROS2的功能包是基于7000端口与控制器通讯，在使用时，首先需要通过winscp进入控制器后台修改控制器配置，在控制器配置文件controller.json中，”hostComputerService”节点下：checkCRC”节点表示是否进行CRC检测，将true改为false。

![](https://ones.inexbot.com/wiki/api/wiki/external-editor/RnqpQ1Yp/UTfgSuen/resources/-UgI8oIE0XYdCeITzIlykh0lXBKwyx3iqEc9fKtBWqI.png)

注意：在修改之前，需要先ssh登陆到控制器

```bash
ssh inexbot@192.168.x.x //ip填控制器的ip
```
密码输入123，即可进行远程控制了。然后，通过以下指令修改权限

```bash
cd robot/
cd config/
sudo chmod 777 controller.json
```
![](https://ones.inexbot.com/wiki/api/wiki/external-editor/RnqpQ1Yp/UTfgSuen/resources/e-CelRBwtDnh6orcimXphV-aqbNs9O-haxeKYKk84rU.png)

修改完权限后，就可以更改controller.json的内容，将“checkCRC”节点中的true改为false，即可使用ROS2的功能包

![](https://ones.inexbot.com/wiki/api/wiki/external-editor/RnqpQ1Yp/UTfgSuen/resources/bRY99mZveCi_eUtD0ji__PJZieikz6kNb7ieaN3bHL0.png)

##### 2.创建工作空间

```bash
mkdir -p ~/inexbot/src
```
##### 3.将下载好的inexbot_service功能包放入/inexbot/src中，然后在inexbot/下，使用colcon build指令编译

```bash
cd inexbot/
colcon build
```
##### 4.编译完成后，通过以下指令，首先运行nrc_rostopic_joint.cpp

```cmake
source install/setup.bash
ros2 run inexbot_service nrc_rostopic_joint
```
若出现终端中以下输出，则代表与控制器连接成功，注意：不同的控制器ip可能不同，需要根据实际情况在nrc_rostopic_joint.cpp中修改ip

![](https://ones.inexbot.com/wiki/api/wiki/external-editor/RnqpQ1Yp/UTfgSuen/resources/zdDP4kHQdb5exXS1Idah_Gbk-wHgMBPd9LbgOa17Ar8.png)

![](https://ones.inexbot.com/wiki/api/wiki/external-editor/RnqpQ1Yp/UTfgSuen/resources/NL4H_0AqMHEvkTD92tvYJYsvLN6wSlaPUZ9TiJ0B6-Q.png)

##### 5.另启动一个终端，同样在inexbot/下，运行rostopic_joint.cpp，若无错误，则效果如下：

```cmake
source install/setup.bash
ros2 run inexbot_service rostopic_joint
```
![](https://ones.inexbot.com/wiki/api/wiki/external-editor/RnqpQ1Yp/UTfgSuen/resources/IQPSbhR94bRpNdLmDE2AIjrQJw0BzwVfts97bbOO0wY.png)

rostopic_joint.cpp作为一个demo，旨在向客户演示ros2与inexbot控制器的通讯。出现input your number字样后，可以输入1-6，不同的数字分别代表了不同的功能。其中，1代表从控制器中读取IO信息，2代表向控制器中写入IO信息，3代表停止查询指令，4代表读取机器人的实际坐标，5，6代表控制机器人走到位姿1和位姿2，相关的代码如下所示：

```cpp
switch(s_number) {
            case 1: // 读IO
            {
                Json::Value rootSend;
                Json::FastWriter fWriter;
                rootSend["channel"] = 1;
                rootSend["stop"] = 0;
                rootSend["robot"] = 1;
                rootSend["mode"] = 1;//为0只回复一次，为定时回复信息
                rootSend["interval"] = 1000;//1000ms回复一次
                rootSend["queryType"][(unsigned int)0] = "IO";
                rootSend["typeCfg"]["IO"][(unsigned int)0] = "DO1";
                rootSend["typeCfg"]["IO"][1] = "DO2";
                rootSend["typeCfg"]["IO"][2] = "DO3";
                rootSend["typeCfg"]["IO"][3] = "DO4";
                rootSend["typeCfg"]["IO"][4] = "DO5";
                str = fWriter.write(rootSend);
                break;
            }
            case 2: // 写IO
            {
                Json::Value rootSend_1;
                Json::FastWriter fWriter_1;
                rootSend_1["IO"]["DO1"] = 1;
                rootSend_1["IO"]["DO2"] = 0;
                rootSend_1["IO"]["DO3"] = 1;
                rootSend_1["IO"]["DO4"] = 0;
                str = fWriter_1.write(rootSend_1);
                break;
            }
            case 3: // 停止查询
            {
                printf("tell me channel number is ???\n");
                int channel_num;
                scanf("%d", &channel_num);
                Json::Value rootSend;
                Json::FastWriter fWriter;
                rootSend["channel"] = channel_num;//需要停止的通道，最大支持9个
                rootSend["stop"] = 1;
                str = fWriter.write(rootSend);
                break;
            }
            case 4: // 读实际坐标
            {
                Json::Value rootSend;
                Json::FastWriter fWriter;
                rootSend["channel"] = 2;
                rootSend["stop"] = 0;
                rootSend["robot"] = 1;
                rootSend["mode"] = 1;
                rootSend["interval"] = 1000;//1000ms回复一次
                rootSend["queryType"][(unsigned int)0] = "realPosMCS";
                str = fWriter.write(rootSend);
                break;
            }
            case 5: // 控制运动
            {
                Json::Value rootSend;
                Json::FastWriter fWriter;
                rootSend["robot"] = 1;
                rootSend["clearBuffer"] = 1;
                rootSend["targetMode"] = 0;
                rootSend["cfg"]["coord"] = "MCS";
                rootSend["cfg"]["speed"] = 20;
                rootSend["cfg"]["acc"] = 20;
                rootSend["cfg"]["moveMode"] = "MOVJ";
                rootSend["targetPos"][0] = 530.433;
                rootSend["targetPos"][1] = 7.497;
                rootSend["targetPos"][2] = 118.728;
                rootSend["targetPos"][3] = 3.141593;
                rootSend["targetPos"][4] = 0;
                rootSend["targetPos"][5] = -0.1;
                str = fWriter.write(rootSend);
                break;
            }
            case 6: // 控制运动
            {
                Json::Value rootSend;
                Json::FastWriter fWriter;
                rootSend["robot"] = 1;
                rootSend["clearBuffer"] = 1;
                rootSend["targetMode"] = 0;
                rootSend["cfg"]["coord"] = "MCS";
                rootSend["cfg"]["speed"] = 20;
                rootSend["cfg"]["acc"] = 20;
                rootSend["cfg"]["moveMode"] = "MOVJ";
                rootSend["targetPos"][0] = 574.964434;
                rootSend["targetPos"][1] = -37.208582;
                rootSend["targetPos"][2] = 150;
                rootSend["targetPos"][3] = 3.141593;
                rootSend["targetPos"][4] = 0;
                rootSend["targetPos"][5] = 0.2;
                str = fWriter.write(rootSend);
                break;
            }
            default:
                RCLCPP_INFO(this->get_logger(), "scanf is: %d", s_number);
                break;
        }
```

# C++ 接口说明

完整接口文档地址：https://doc.hmilib.inexbot.coision.cn/nrc__interface_8h.html

## 核心连接接口

| 函数 | 说明 |
|------|------|
| `SOCKETFD connect_robot(const char* ip, const char* port)` | 连接到控制器 |
| `int get_connection_status(SOCKETFD fd)` | 获取连接状态 |
| `void disconnect_robot(SOCKETFD fd)` | 断开连接 |

## 位置获取接口

| 函数 | 说明 |
|------|------|
| `int get_current_position(SOCKETFD fd, int coord, std::vector<double>& pos)` | 获取当前位置 |
| `int get_current_extra_position(SOCKETFD fd, std::vector<double>& pos)` | 获取外部轴位置 |

**coord 参数说明：**
- 0: 关节坐标
- 1: 直角坐标

### MoveCmd 结构体

```cpp
struct MoveCmd {
    int coord;                      // 坐标系 (0-关节, 1-直角)
    PosType targetPosType;           // 目标位置类型
    std::vector<double> targetPosValue; // 目标位置值
    double velocity;                // 速度
    double acc;                     // 加速度
    double dec;                     // 减速度
    double pl;                      // 平滑度
};
```

## 伺服跟踪运动

| 函数 | 说明 |
|------|------|
| `int servo_move(SOCKETFD fd, ServoMovePara servoMove)` | 伺服跟踪运动 |
| `int servo_point_position_motion_control(SOCKETFD fd, ServoPointMovePara servoMove)` | 伺服点位位置控制 |
| `int enable_servo_position_motion_control(SOCKETFD fd, int enable)` | 开启/关闭伺服控制模式 |

### ServoMovePara 结构体

```cpp
struct ServoMovePara {
    bool clearBuffer;                           // 是否清除之前发送的未开始插补计算的点位
    int targetMode;                             // 0-独立点 1-连续轨迹
    int sendMode;                               // 0-一次传输完全部轨迹 1-一次传输部分点位
    int runMode;                                // 0-接收完再运动 1-边接受边运动
    int sum;                                    // 总传输次数
    int count;                                  // 当前是第几次传输
    int coord;                                  // 0-关节 1-直角
    int size;                                   // 本次传输的点位数
    std::vector<std::vector<double>> pos;      // 二维数组，目标点位
    std::vector<std::vector<double>> axisvel;  // 每个轴的速度
    std::vector<std::vector<double>> axisacc; // 每个轴的加速度
    std::vector<double> timeStamp;              // 到达该点位的时间，单位ms
};
```

### ServoPointMovePara 结构体

```cpp
struct ServoPointMovePara {
    bool end;                                  // 是否结束
    int sum;                                   // 总共要发的帧数
    int count;                                 // 当前为第几帧
    std::vector<std::vector<double>> pos;      // 前7位为机器人本体，后5位为外部轴
};
```

## servoJ 关节跟踪模式

| 函数 | 说明 |
|------|------|
| `Result open_servoJ(SOCKETFD fd, std::vector<double> vmax, std::vector<double> amax, std::vector<double> jmax)` | 打开关节跟踪模式 |
| `Result set_servoJ_pos(SOCKETFD fd, std::vector<double> q)` | 发送跟踪关节位置 |
| `Result stop_servoJ(SOCKETFD fd)` | 关闭关节跟踪模式 |

**参数说明：**
- vmax: 速度约束，单位 度/秒
- amax: 加速度约束，单位 度/秒²
- jmax: 加加速度约束，单位 度/秒³
- q: 目标位置，单位 度

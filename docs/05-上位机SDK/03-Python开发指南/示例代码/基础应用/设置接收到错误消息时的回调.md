# Python 接口说明

Python SDK 使用方式与 C++ SDK 类似，通过 `nrc_interface` 模块调用。

## 核心连接接口

| 函数 | 说明 |
|------|------|
| `nrc_interface.connect_robot(ip, port)` | 连接到控制器 |
| `nrc_interface.get_connection_status(fd)` | 获取连接状态 |
| `nrc_interface.VectorDouble(size)` | 创建双精度向量 |
| `nrc_interface.VectorVectorDouble()` | 创建双精度向量数组 |

## 位置获取接口

| 函数 | 说明 |
|------|------|
| `nrc_interface.get_current_position(fd, coord, pos)` | 获取当前位置 |
| `nrc_interface.queue_motion_set_status(fd, status)` | 设置队列状态 |

## 运动控制接口

| 函数 | 说明 |
|------|------|
| `nrc_interface.MoveCmd()` | 创建运动指令 |
| `nrc_interface.queue_motion_push_back_moveJ(fd, cmd)` | 添加关节运动到队列 |
| `nrc_interface.queue_motion_send_to_controller(fd, count)` | 发送队列到控制器 |

### MoveCmd 属性

| 属性 | 说明 |
|------|------|
| `coord` | 坐标系 (0-关节, 1-直角) |
| `targetPosType` | 目标位置类型 |
| `targetPosValue` | 目标位置值 |
| `velocity` | 速度 |
| `acc` | 加速度 |
| `dec` | 减速度 |
| `pl` | 平滑度 |

### PosType_data

```python
moveCmd.targetPosType = nrc_interface.PosType_data
```

## servo_move 伺服跟踪运动

| 函数 | 说明 |
|------|------|
| `nrc_interface.ServoMovePara()` | 创建伺服运动参数 |
| `nrc_interface.servo_move(fd, servoMove)` | 伺服跟踪运动 |

### ServoMovePara 属性

| 属性 | 说明 |
|------|------|
| `robotNum` | 控制第几个机器人 |
| `clearBuffer` | 是否清除之前发送的未开始插补计算的点位 |
| `targetMode` | 0-独立点 1-连续轨迹 |
| `sendMode` | 0-一次传输完全部轨迹 1-一次传输部分点位 |
| `runMode` | 0-接收完再运动 1-边接受边运动 |
| `sum` | 总传输次数 |
| `count` | 当前是第几次传输 |
| `coord` | 0-关节 1-直角 |
| `size` | 本次传输的点位数 |
| `pos` | 二维数组，目标点位 |
| `axisvel` | 每个轴的速度 |
| `axisacc` | 每个轴的加速度 |
| `timeStamp` | 到达该点位的时间，单位ms |

## 回调函数

| 函数 | 说明 |
|------|------|
| `nrc_interface.set_receive_error_or_warnning_message_callback(fd, callback)` | 设置错误/警告消息回调 |

### 回调函数格式

```python
def my_callback(message_type: int, message: str, message_code: int):
    print(f"[回调] 类型={message_type}, 信息={message}, 代码={message_code}")
```

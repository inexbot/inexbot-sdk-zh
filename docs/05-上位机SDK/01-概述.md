# 上位机SDK概述

libnrc_host 是一个 C++ 库，提供了一套可以快速集成到项目中的网络接口（基于 Socket_API 通讯协议封装），让开发者无需关注底层协议细节，提高开发效率。

## 兼容性说明

支持 Linux、Windows 平台，以及多种高级开发语言，方便开发者选择最适合自己的平台进行开发。

| 类别 | 兼容范围 |
|------|----------|
| 系统 | 支持 Windows\Linux，x86、x86_64、arm、arm64 架构 |
| 开发环境 | 针对自己的开发语言使用对应的 QT,Python,C# 推荐的 IDE，如 Qtcreator, VS Studio，VSCode 等 |

## SDK 目录结构

```
SDK/
├── C++/
│   ├── c_interface/          # C接口
│   ├── cpp_interface/        # C++接口
│   ├── parameter/            # 数据结构定义
│   ├── win_mingw64_v1.x.x/   # Windows MinGW 64位 Release库
│   ├── win_mingw64_debug_v1.x.x/  # Windows MinGW 64位 Debug库
│   ├── win_msvc64_v1.x.x/    # Windows MSVC 64位 Release库
│   ├── win_msvc64_debug_v1.x.x/  # Windows MSVC 64位 Debug库
│   ├── linux_x86_v1.x.x/     # Linux x86 Release库
│   ├── linux_x86_debug_v1.x.x/   # Linux x86 Debug库
│   ├── linux_aarch64_v1.x.x/ # Linux aarch64 Release库
│   └── linux_aarch64_debug_v1.x.x/ # Linux aarch64 Debug库
├── Python/
│   └── nrc_host.pyd/.so      # Python动态链接库
├── C#/
│   ├── Csharp_api/           # C#接口文件
│   ├── win_csharp_v1.x.x/     # Windows .NET Release库
│   └── win_csharp_debug_v1.x.x/ # Windows .NET Debug库
```

## 支持的开发语言

- **C++**: 支持 Windows (MinGW/MSVC) 和 Linux (x86/aarch64)
- **Python**: 需要与编译库版本匹配的Python版本
- **C#**: 支持 .NET Framework 4.8 和 .NET 8+

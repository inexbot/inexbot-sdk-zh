# C# 接口说明

C# SDK 通过 nrc_interface 模块调用，提供与控制器通信的接口。

## 核心连接接口

| 函数 | 说明 |
|------|------|
| `nrc_interface.connect_robot(ip, port)` | 连接到控制器 |
| `nrc_interface.recv_message(fd, callback)` | 设置消息接收回调 |

## 回调函数

C# 使用委托来处理消息接收，需要使用 `UnmanagedFunctionPointer` 特性。

```csharp
[UnmanagedFunctionPointer(CallingConvention.Cdecl)]
public delegate void RecvMsgCallbackDelegate(int cmd, IntPtr message);
```

### SWIGTYPE_p_f_int_p_q_const__char__void

这是 SWIG 生成的回调函数类型，用于接收消息。

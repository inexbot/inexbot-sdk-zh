## 📖 关于纳博特科技

纳博特科技成立于 2015 年，是中国高端运动控制领域的技术先锋。我们以“软件定义控制、算法驱动智能”为核心理念，提供从底层硬件、中间件平台到上层应用的全栈式技术体系。

我们的核心技术涵盖：
*   **多轴高精度运动控制** (基于 PC-Based 架构)
*   **机器人核心算法** (运动学/动力学建模、在线轨迹规划)
*   **智能感知与协同** (2D/3D 视觉融合、力控、多智能体协同)
*   **行业工艺包** (焊接、切割、点胶、打磨、喷涂等)

我们的产品已广泛应用于工业机器人、半导体、医疗、物流及新能源汽车等领域，服务于比亚迪、富士康、立讯精密等全球知名客户。

## 🚀 核心产品与技术亮点

### 1. 控制系统系列
*   **C2200 控制器**: 采用 IPC+RTOS 架构，全国产 ARM 方案 + Linux RT 实时系统。支持最多 64 轴同步运动，内置丰富的工艺包（焊接、切割、码垛等）。
*   **X01 星辰一号**: 专为协作机器人设计，集成基于动力学的拖拽示教、碰撞检测及动力学前馈功能，即插即用。

### 2. NexDroid 开放平台
利用 **NexDroidAPI**，开发者可以结合行业经验快速开发新型自动化装备。
*   支持多种机器人构型（串联 6 轴、SCARA、并联、直角坐标等）。
*   提供强大的二次开发能力，支持自定义运动学与动力学模型。
*   适用于 3C 电子、医疗、核能、工程机械等多个行业。

### 3. 智能工艺包
预置了经过大规模验证的行业工艺系统：
*   **焊接**: 汽车点焊、钢构弧焊、激光焊接。
*   **加工**: 三维五轴激光切割、打磨抛光。
*   **精密**: 高精度点胶、晶圆搬运。
